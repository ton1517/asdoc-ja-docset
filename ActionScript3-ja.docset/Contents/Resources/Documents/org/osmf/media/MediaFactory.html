クラスpublic class MediaFactory

サブクラス DefaultMediaFactory

言語バージョン: ActionScript 3.0
製品バージョン: OSMF 1.0
ランタイムバージョン: Flash Player 10, AIR 1.5

MediaFactory は、メディアエレメントのファクトリクラスを表します。

ファクトリの操作では、MediaResourceBase を入力して伴い、MediaElement を出力として生成します。

MediaFactory は、MediaFactoryItem オブジェクトのリストを保持します。各 MediaFactoryItem オブジェクトは、特定の MediaElement の作成に必要なすべての情報をカプセル化します。MediaFactory は、各 MediaFactoryItem の canHandleResourceFunction メソッドに依存して、指定された MediaResourceBase を処理可能な MediaFactoryItem オブジェクトを探します。

また、ファクトリインターフェイスは、特定の MediaFactoryItem オブジェクトに対してクエリを実行するメソッドや、（MediaFactoryItem オブジェクトを保持する）プラグインをロードするメソッドを公開します。

パラメーター

例外
ArgumentError — 引数が null である場合、または引数が null の ID フィールドを持つ場合。

loadPlugin

()メソッド 
public function loadPlugin(resource:MediaResourceBase):void

言語バージョン: ActionScript 3.0
製品バージョン: OSMF 1.0
ランタイムバージョン: Flash Player 10, AIR 1.5

指定されたリソースで特定されるプラグインをロードします。プラグインが既にロードされている場合、MediaFactory はプラグインを再ロードしません。正常なロード時には、プラグインの PluginInfo プロパティ内の MediaFactoryItem がこの MediaFactory に追加され、MediaFactoryEvent.PLUGIN_LOAD イベントが送出されます。それ以外の場合（ロードに失敗した場合）、MediaFactoryEvent.PLUGIN_LOAD_ERROR イベントが送出されます。

パラメーター

resource:MediaResourceBase — プラグインを表す MediaResourceBase です。リモート（動的）プラグインの場合、ロードするリモート SWF を示す URLResource を使用します。ローカル（静的）プラグインの場合、PluginInfoResource を使用します。

例外
ArgumentError — resource が null である場合、または resource が URLResource と PluginInfoResource のいずれでもない場合。

resolveItems

()メソッド 
protected function resolveItems(resource:MediaResourceBase, items:Vector.<MediaFactoryItem>):MediaFactoryItem

指定されたリストの MediaFactoryItem の範囲外で、指定されたリソースに最適な MediaFactoryItem を返します。複数の MediaFactoryItem が処理可能なリソースで createMediaElement が呼び出された場合、このメソッドが呼び出されます。サブクラスでオーバーライドして、最適なものを選択できます。デフォルトの動作では、プラグインは主要なメディアタイプよりも優先されるべきであるという理論から、フレームワークに「ネイティブ」でない最初のアイテムが選択されます。この決定は、「org.osmf」で始まる ID 値の有無に基づいて行われます。

例 この例の使用方法
MediaFactoryExample.as

この例は、MediaFactory を使用してプラグインをロードする方法を示しています。サンプルでは、モックアップのプラグインが URL で使用されていることに注意してください。
コピー

package org.osmf.media
{
    import flash.display.Sprite;
    
    import org.osmf.events.MediaFactoryEvent;

    public class MediaFactoryExample extends Sprite
    {
        public function MediaFactoryExample()
        {
            // Construct a media factory, and listen to its plug-in related events:
            var factory:MediaFactory = new MediaFactory();
            factory.addEventListener(MediaFactoryEvent.PLUGIN_LOAD, onPluginLoaded);
            factory.addEventListener(MediaFactoryEvent.PLUGIN_LOAD_ERROR, onPluginLoadError);
            
            // Instruct the factory to load the plug-in at the given url:
            factory.loadPlugin(new URLResource("http://myinvalidurl.com/foo.swf"));        
        }
    
        private function onPluginLoaded(event:MediaFactoryEvent):void
        {
            // Use the factory to create a media-element related to the plugin:
            var factory:MediaFactory = event.target as MediaFactory;
            factory.createMediaElement(new URLResource("http://myinvalidurl.com/content"));
        }
        
        private function onPluginLoadError(event:MediaFactoryEvent):void
        {
            // Handle plug-in loading failure:
            trace("Plugin failed to load.");    
        }    
    }
}
