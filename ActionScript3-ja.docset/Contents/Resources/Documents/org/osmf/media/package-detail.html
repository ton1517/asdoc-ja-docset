org.osmf.media パッケージには、メディアの再生に使用するクラス、およびメディアの再生のための下位レベルの構築ブロックが含まれます。


クラス
 クラス説明
 DefaultMediaFactory DefaultMediaFactory は、MediaFactory のデフォルト実装です。
 LoadableElementBase LoadableElementBase は、LoadTrait を持つメディアエレメントの基本クラスです。
 MediaElement MediaElement は、統合メディアエクスペリエンスを表します。
 MediaFactory MediaFactory は、メディアエレメントのファクトリクラスを表します。
 MediaFactoryItem MediaFactoryItem は、MediaFactory から MediaElement を動的に作成して初期化するために必要なすべての情報がカプセル化されたものです。
 MediaFactoryItemType MediaFactoryItemType は、MediaFactoryItem の有効なタイプの列挙です。
 MediaPlayer MediaPlayer は、すべてのメディアタイプを操作するために使用されるコントローラークラスです。
 MediaPlayerSprite MediaPlayerSprite は、MediaPlayer、MediaContainer および MediaFactory の機能を Sprite ベースの一体型のクラスで提供します。
 MediaPlayerState MediaPlayerState クラスは、MediaPlayer の現在の状態を示すパブリック定数を列挙します。
 MediaResourceBase MediaResourceBase は、MediaElement への入力として機能するメディアの基本クラスです。
 MediaType MediaType は、Open Source Media Framework ですぐに処理できる各種のメディアタイプを識別します。
 PluginInfo PluginInfo は、プラグインが正常にロードされた後にアプリケーションで使用可能な MediaFactoryItem オブジェクトがカプセル化されたものです。
 PluginInfoResource PluginInfoResource は、静的プラグインを表すメディアリソースです。
 URLResource URLResource は、URL を持つメディアリソースです。
