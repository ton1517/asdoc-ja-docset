クラスpublic class NetLoader
継承NetLoader LoaderBase EventDispatcher Object
サブクラス HTTPStreamingNetLoader, MulticastNetLoader, RTMPDynamicStreamingNetLoader

言語バージョン: ActionScript 3.0
製品バージョン: OSMF 1.0
ランタイムバージョン: Flash Player 10, AIR 1.5

NetLoader クラスは LoaderBase を拡張し、AudioElement クラスと VideoElement クラスにロード機能を提供します。

ストリーミングとプログレッシブの両方のメディアリソースがサポートされます。リソース URL が RTMP の場合、NetConnectionFactoryBase を呼び出して RTMP サーバーに接続します。NetConnection は LoadTrait インスタンス間で共有できます。リソース URL が HTTP の場合、プログレッシブダウンロードでは connect(null) が実行されます。

NetLoader では、NetConnection 経由で認証トークンを渡す場合に、Flash メディアトークン認証を使用できます。

プロパティの詳細

reconnectTimeout

プロパティ
reconnectTimeout:Number

ストリームの再接続タイムアウト（ミリ秒）です。

この期間内に再接続に成功しなければ、NetLoader はストリームの再接続をあきらめます。 デフォルトは 120 秒です。一時停止しないストリームの場合、タイムアウトの期間は、バッファーが空になったときから始まるので、ゼロ秒という値は有効で、バッファーが空になったら再接続を試みないという意味になります。一時停止するストリームの場合、タイムアウトの期間は直ちに開始します。

パラメーター
factory:NetConnectionFactoryBase (default = null) — NetConnection の管理に使用する NetConnectionFactoryBase インスタンスです。factory が null の場合、NetConnectionFactory が作成されて使用されます。NetConnectionFactory クラスは接続の共有を容易にします。これによりグローバル共有を簡単に設定できます。1 つの NetConnectionFactory インスタンスをプレーヤー内に作成し、すべての NetLoader インスタンスに渡します。

reconnect

()メソッド 
protected function reconnect(netConnection:NetConnection, resource:URLResource):void

指定された NetConnection を、指定された URL に再接続しようとします。

ストリーム再接続機能の使用時に、クライアントは、このメソッドをオーバーライドして、NetConnection のカスタム動作を実現できます。 例えば、クライアント側のロードバランシングをプレーヤーで実現する必要がある場合には、カスタム NetLoader クラスを作成し、このメソッドをオーバーライドして代替 URI を使用できます。

パラメーター

netConnection:NetConnection — ストリームの再接続ロジックによって作成された新しい NetConnection です。
 
resource:URLResource — メディアの再生に最初に使用された URLResource です。
例 この例の使用方法
NetLoaderExample.as

コピー
package
{
    import flash.display.Sprite;
    import flash.display.StageAlign;
    import flash.display.StageScaleMode;
    
    import org.osmf.elements.VideoElement;
    import org.osmf.media.MediaPlayerSprite;
    import org.osmf.media.URLResource;
    import org.osmf.net.NetLoader;
    
    public class NetLoaderExample extends Sprite
    {
        public function NetLoaderExample()
        {
            super();
            
            stage.scaleMode = StageScaleMode.NO_SCALE;
            stage.align = StageAlign.TOP_LEFT;
            
            var mediaPlayerSprite:MediaPlayerSprite = new MediaPlayerSprite();
            var netLoader:NetLoader = new NetLoader();
            var urlResource:URLResource = new URLResource("rtmp://cp67126.edgefcs.net/ondemand/mediapm/strobe/content/test/SpaceAloneHD_sounas_640_500_short");
            var videoElement:VideoElement = new VideoElement(urlResource, netLoader);
            
            addChild(mediaPlayerSprite);
            mediaPlayerSprite.media = videoElement;    
        }    
    }
}
