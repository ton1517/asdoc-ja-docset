public function StreamingItem(type:String, streamName:String, bitrate:Number = 0, info:Object = null)

言語バージョン: ActionScript 3.0
製品バージョン: OSMF 1.6
ランタイムバージョン: Flash Player 10, AIR 1.5

デフォルト・コンストラクター

パラメーター
type:String — ストリームの StreamingItemType です。 指定できる値は、VIDEO（ビデオを含むストリームの場合）および AUDIO（オーディオのみを含むストリームの場合）です。
 
streamName:String — このストリームを特定するために使用する String です。
 
bitrate:Number (default = 0) — ストリームのエンコード後のビットレートを 1 秒あたりのキロビット数（kbps）で指定する Number です。
 
info:Object (default = null) — ストリームに関連付けられているカスタム情報が含まれる Object です。 一般にこの中には、ビデオの幅と高さが含まれますが、ユーザーにわかりやすいストリームの説明が含まれる場合もあります。
