クラスpublic class ImageElement
継承ImageElement LoadableElementBase MediaElement EventDispatcher Object

言語バージョン: ActionScript 3.0
製品バージョン: OSMF 1.0
ランタイムバージョン: Flash Player 10, AIR 1.5

ImageElement は、静止イメージの表示用に特別に作成されるメディアエレメントです。PNG、GIF、JPG などのイメージをロードして表示できます。

ImageElement を作成して使用する基本的な手順は、次のとおりです。

  1. ロードするイメージの URL を指し示す、新規の URLResource を作成します。
  2. 新規の ImageElement を作成し、パラメーターとして URLResource を渡します。
  3. 新規の MediaPlayer を作成します。
  4. ImageElement を MediaPlayer の media プロパティに割り当てます。
  5. MediaPlayer の displayObject プロパティから DisplayObject を取得し、表示リストに追加します。displayObject プロパティをすぐに取得できない場合は、MediaPlayer の displayObjectChange イベントを監視します。
  6. ImageElement の作業が終了したら、MediaPlayer の media プロパティを null に設定し、表示リストから DisplayObject を削除します。

例 この例の使用方法
ImageElementExample.as

コピー
package
{
    import flash.display.Sprite;
    import flash.display.StageAlign;
    import flash.display.StageScaleMode;
    
    import org.osmf.elements.ImageElement;
    import org.osmf.elements.VideoElement;
    import org.osmf.media.MediaPlayer;
    import org.osmf.media.MediaPlayerSprite;
    import org.osmf.media.URLResource;
    
    public class ImageElementExample extends Sprite
    {
        public function ImageElementExample()
        {
            super();
            
            stage.scaleMode = StageScaleMode.NO_SCALE;
            stage.align = StageAlign.TOP_LEFT;
            
            var mediaPlayerSprite:MediaPlayerSprite = new MediaPlayerSprite();
            var imageElement:ImageElement = new ImageElement();
            imageElement.resource = new URLResource("http://mediapm.edgesuite.net/strobe/content/test/train.jpg");
            
            addChild(mediaPlayerSprite);
            mediaPlayerSprite.media = imageElement;    
        }
    }
}

ActionScript 3.0 リファレンスガイドのコンテンツが英語で表示されます。

ActionScript 3.0 リファレンスガイドのすべての部分がすべての言語に翻訳されているわけではありません。言語エレメントが翻訳されていない場合、そのエレメントは英語で表示されます。例えば、ga.controls.HelpBox クラスはどの言語にも訳されていません。このため、リファレンスガイドの日本語バージョンでは、ga.controls.HelpBox クラスは英語で表示されます。
