Metadata クラスは、メタデータの関連コレクションをカプセル化します。

Metadata は、キーと値のペアで構成されます。キーは String、値は任意の Object です。Metadata クラスは、これらのキーと値のペアを操作するための厳密に型指定された API と、メタデータに対する変更を検出するイベントを提供します。

addValue

()メソッド
public function addValue(key:String, value:Object):void

言語バージョン: ActionScript 3.0
製品バージョン: OSMF 1.0
ランタイムバージョン: Flash Player 10, AIR 1.5

指定されたキーを使用して、指定された値をこの Metadata オブジェクトに保存します。キーは、引き続き値の取得に使用できます。キーが、既に Metadata オブジェクト内に存在する別のオブジェクトのキーと同じである場合、この関連先は新しい値で上書きされます。

パラメーター

key:String — 値を関連付けるキーです。
 
value:Object — Metadata オブジェクトに追加する値です。

例外
ArgumentError — key が null または無効である場合。
