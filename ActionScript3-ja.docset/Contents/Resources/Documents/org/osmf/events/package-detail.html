org.osmf.events パッケージには、OSMF イベントオブジェクトを定義するクラスが含まれています。


クラス
 クラス説明
 AlternativeAudioEvent AlternativeAudioEvent は、AlternativeAudioTrait のプロパティが変更されたときに送出されます。
 AudioEvent AudioEvent は、AudioTrait のプロパティが変更されたときに送出されます。
 BufferEvent BufferEvent は、BufferTrait のプロパティが変更されたときに送出されます。
 ContainerChangeEvent ContainerChangeEvent は、IMediaContainer への参照が変更されたときに送出されます。
 DisplayObjectEvent DisplayObjectEvent は、DisplayObjectTrait のプロパティが変更されたときに送出されます。
 DRMEvent DRMEvent は、DRMTrait のプロパティが変更されたときに送出されます。
 DVREvent DVREvent は、DVRTrait のプロパティが変更されたときに送出されます。
 DynamicStreamEvent DynamicStreamEvent は、DynamicStreamTrait のプロパティが変更されたときに送出されます。
 LoaderEvent LoaderBase でロード中またはアンロード中の LoadTrait で、ロード指向の大きな変更が発生すると、LoaderBase によって LoaderEvent が送出されます。
 LoadEvent LoadEvent は、LoadTrait のプロパティが変更されたときに送出されます。
 MediaElementEvent MediaElementEvent は、MediaElement のプロパティが変更されたときに送出されます。
 MediaError MediaError は、MediaElement に関連するエラーをカプセル化します。
 MediaErrorCodes MediaErrorCodes クラスは、エラー ID の静的定数を提供します。
 MediaErrorEvent MediaElement でエラーが発生すると、MediaElement によって MediaErrorEvent が送出されます。
 MediaFactoryEvent MediaFactoryEvent は、MediaFactory によって、MediaElement が作成されたとき、またはプラグインのロードが成功または失敗したときに送出されます。
 MediaPlayerCapabilityChangeEvent MediaPlayer の機能が変更されると、MediaPlayer によって MediaPlayerCapabilityChangeEvent が送出されます。
 MediaPlayerStateChangeEvent このイベントは、MediaPlayer の state プロパティが変更されたときに MediaPlayer によって送出されます。
 MetadataEvent メタデータ値が追加、削除または変更されると、Metadata オブジェクトによって MetadataEvent が送出されます。
 MetricError MetricError は、アダプティブビットレートアルゴリズムで使用するメトリックに関連したエラーをカプセル化したオブジェクトです。
 MetricErrorCodes MetricErrorCodes クラスは、エラー ID の静的定数を提供します。
 NetConnectionFactoryEvent NetConnectionFactory によって NetConnection の確立が成功または失敗すると、NetConnectionFactory によって NetConnectionFactoryEvent が送出されます。
 PlayEvent PlayEvent は、PlayTrait のプロパティが変更されたときに送出されます。
 QoSInfoEvent NetStream は、新しい QoSInfo オブジェクトを生成したときに QoSInfoEvent を送出します。
 SeekEvent SeekEvent は、SeekTrait のプロパティが変更されたときに送出されます。
 SerialElementEvent SerialElementEvent は、SerialElement のプロパティが変更されたときに送出されます。
 TimeEvent TimeEvent は、TimeTrait のプロパティが変更されたときに送出されます。
 TimelineMetadataEvent TimelineMetadataEvent は、TimelineMetadata オブジェクトのプロパティが変更されたときに送出されます。
