クラスpublic class MediaError
継承MediaError Error Object

言語バージョン: ActionScript 3.0
製品バージョン: OSMF 1.0
ランタイムバージョン: Flash Player 10, AIR 1.5

MediaError は、MediaElement に関連するエラーをカプセル化します。エラーは、エラー ID と対応するメッセージで表されます。エラー ID の 0 から 999 は、フレームワークが使用するために予約されています。

発生する可能性があるすべてのフレームワークレベルのエラーのリストは、MediaErrorCodes クラスで確認できます。

カスタムエラーの場合、カスタムエラーのメッセージを返すには、クライアントで MediaError をサブクラス化し、getMessageForErrorID をオーバーライドする必要があります。

パラメーター
errorID:int — エラーの ID です。対応するメッセージを調べる場合に使用します。エラー ID の 0 から 999 は、フレームワークが使用するために予約され、MediaErrorCodes に定義されています。
 
detail:String (default = null) — オプションのストリングで、エラーをサポートする詳細が含まれます。通常、このストリングは、単に Flash Player API. で提供されるエラー詳細です。

getMessageForErrorID

()メソッド
protected function getMessageForErrorID(errorID:int):String

言語バージョン: ActionScript 3.0
製品バージョン: OSMF 1.0
ランタイムバージョン: Flash Player 10, AIR 1.5

指定された ID のエラーのメッセージを返します。エラー ID が不明であった場合は、空のストリングが返されます。

このメソッドでは message プロパティで公開されている値が返されるので、カスタムエラーの場合は、サブクラスでオーバーライドしてメッセージを提供する必要があります。

パラメーター

errorID:int — エラーの ID です。

戻り値
String — 指定された ID のエラーのメッセージです。
