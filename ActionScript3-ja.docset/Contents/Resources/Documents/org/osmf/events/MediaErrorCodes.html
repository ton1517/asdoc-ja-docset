パッケージorg.osmf.events
クラスpublic final class MediaErrorCodes
継承MediaErrorCodes Object

言語バージョン: ActionScript 3.0
製品バージョン: OSMF 1.0
ランタイムバージョン: Flash Player 10, AIR 1.5

MediaErrorCodes クラスは、エラー ID の静的定数を提供します。エラー ID の 0 から 999 は、フレームワークが使用するために予約されています。
