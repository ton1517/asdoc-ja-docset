パッケージfl.motion
クラスpublic class MatrixTransformer
継承MatrixTransformer Object

言語バージョン: ActionScript 3.0
製品バージョン: Flash CS3
ランタイムバージョン: Flash Player 9.0.28.0, AIR 1.0

MatrixTransformer クラスには、変換マトリックスの個々のプロパティ（水平/垂直スケール、水平/垂直傾斜角度、回転など）を変更するためのメソッドが存在します。このクラスには、通常の（0, 0) 座標ではなく、特定の変換ポイントを軸として回転させるためのメソッドも用意されています。

public static function getRotation(m:Matrix):Number

言語バージョン: ActionScript 3.0

ランタイムバージョン: Flash Player 9.0.28.0, AIR 1.0

マトリックス内の回転角度を計算します（度単位）。水平傾斜角度と垂直傾斜角度が異なる場合は、垂直傾斜角度の値が使用されます。これは、Flash Player の回転ビヘイビアーと一致します。

パラメーター

m:Matrix — Matrix インスタンス。

戻り値
Number — 回転角度（度単位）。

matchInternalPointWithExternal

()メソッド 
public static function matchInternalPointWithExternal(m:Matrix, internalPoint:Point, externalPoint:Point):void

言語バージョン: ActionScript 3.0
製品バージョン: Flash CS3
ランタイムバージョン: Flash Player 9.0.28.0, AIR 1.0

内部のポイントと外部のポイントとが一致するようにマトリックスを移動します。これを使用して、変換後のムービークリップ内のポイントを、その親内のポイントに合わせることができます。

パラメーター

m:Matrix — Matrix インスタンス。
 
internalPoint:Point — マトリックスの変換空間内の位置を定義する Point インスタンス。
 
externalPoint:Point — マトリックスの変換空間外の基準位置を定義する Point インスタンス。

関連する API エレメント

flash.geom.Matrix

rotateAroundExternalPoint

()メソッド 
public static function rotateAroundExternalPoint(m:Matrix, x:Number, y:Number, angleDegrees:Number):void

言語バージョン: ActionScript 3.0
製品バージョン: Flash CS3
ランタイムバージョン: Flash Player 9.0.28.0, AIR 1.0

マトリックスの変換空間外に定義されたポイントを軸としてマトリックスを回転します。これにより、親内の変換ポイントを軸としてムービークリップを回転させることができます。

パラメーター

m:Matrix — Matrix インスタンス。
 
x:Number — ポイントの x 座標です。
 
y:Number — ポイントの y 座標です。
 
angleDegrees:Number — 回転角度（度単位）。

関連する API エレメント

setRotation

()メソッド 
public static function setRotation(m:Matrix, rotation:Number):void

言語バージョン: ActionScript 3.0
製品バージョン: Flash CS3
ランタイムバージョン: Flash Player 9.0.28.0, AIR 1.0

マトリックス内の回転角度を変更します。水平傾斜角度と垂直傾斜角度が異なる場合、垂直傾斜角度は rotation 値に設定され、水平傾斜角度には、新旧の rotation 値の差が加算されます。これは、Flash Player の回転ビヘイビアーと一致します。

パラメーター

m:Matrix — Matrix インスタンス。
 
rotation:Number — 回転角度（度単位）。
