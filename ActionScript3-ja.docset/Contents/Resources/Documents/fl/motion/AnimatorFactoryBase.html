ランタイムバージョン: Flash Player 9.0.28.0, AIR 1.0

AnimatorFactoryBase クラスでは、実行時に 1 つの Motion で複数のターゲットオブジェクトを動的に表示およびトゥイーンする ActionScript ベースのサポートが提供されています。 AnimatorFactoryBase は、AnimatorBase クラスを使用して、1 つの Motion（MotionBase から派生）を複数のトゥイーンインスタンス（ターゲットオブジェクト）に割り当てます。ここで、AnimatorBase クラスは、単一のターゲットトゥイーンオブジェクトを持つ単一の Motion インスタンスに関連付けられます。 AnimatorFactoryBase クラス自体は使用しないでください。 代わりに、サブクラス（AnimatorFactory または AnimatorFactory3D）を使用してください。

addTarget

public function addTarget(target:DisplayObject, repeatCount:int = 0, autoPlay:Boolean = true, startFrame:int = -1, useCurrentFrame:Boolean = false):AnimatorBase

言語バージョン: ActionScript 3.0
製品バージョン: Flash CS4
ランタイムバージョン: Flash Player 9.0.28.0, AIR 1.0

AnimatorBase インスタンスを作成し、返します。このインスタンスの target プロパティは、該当する場合、DisplayObject（targetParent の targetName プロパティ）に設定され、その Motion プロパティは、作成時に AnimatorFactoryBase インスタンスに格納されます。

パラメーター

target:DisplayObject — モーショントゥイーンを使用する表示オブジェクトです。
 
repeatCount:int (default = 0) — アニメーションを再生する回数です。デフォルト値の 0 では、アニメーションが無限にループします。
 
autoPlay:Boolean (default = true) — アニメーションの再生を自動的に開始するかどうかを指定する値（デフォルトは true）です。
 
startFrame:int (default = -1) — 親のタイムラインを基準として、アニメーションが開始するフレームです。親のタイムラインが関連付けられたモーションの継続時間より短い場合、startFrame はこの addTarget 呼び出しが行われてターゲットアニメーションが開始する前に、フレームの数を示します。
 
useCurrentFrame:Boolean (default = false) — true の場合、ターゲットオブジェクトを配置するアニメーションフレームを決定する親の currentFrame プロパティの使用を指定するフラグです。

戻り値
AnimatorBase — 新しい AnimatorBase インスタンスです。

public function addTargetInfo(targetParent:DisplayObject, targetName:String, repeatCount:int = 0, autoPlay:Boolean = true, startFrame:int = -1, useCurrentFrame:Boolean = false, initialPosition:Array = null, zIndex:int = -1, placeholderName:String = null, instanceFactoryClass:Class = null):AnimatorBase

言語バージョン: ActionScript 3.0
製品バージョン: Flash CS4
ランタイムバージョン: Flash Player 9.0.28.0, AIR 1.0

親 DisplayObjectContainer を参照し、AnimatorBase インスタンスを作成して返します。このインスタンスの target プロパティは、該当する場合、DisplayObject に設定されます。これは、targetParent の targetName プロパティで、Motion プロパティは作成時に AnimatorFactoryBase インスタンスに格納されます。

パラメーター

targetParent:DisplayObject — 親 DisplayObjectContainer です。
 
targetName:String — 親を参照するターゲットインスタンス名です。
 
repeatCount:int (default = 0) — アニメーションを再生する回数です。デフォルト値の 0 では、アニメーションが無限にループします。
 
autoPlay:Boolean (default = true) — アニメーションの再生を自動的に開始するかどうかを指定する値（デフォルトは true）です。
 
startFrame:int (default = -1) — 親のタイムラインを基準として、アニメーションが開始するフレームです。親のタイムラインが関連付けられたモーションの継続時間より短い場合、startFrame はこの addTarget 呼び出しが行われてターゲットアニメーションが開始する前に、フレームの数を示します。親が SimpleButton の場合は、startFrame を使用して、モーションが適用されるボタンの状態（0 は upState、1 は overState、2 は downState）を示します。
 
useCurrentFrame:Boolean (default = false) — true の場合、ターゲットオブジェクトを配置するアニメーションフレームを決定する親の currentFrame プロパティの使用を指定するフラグです。
 
initialPosition:Array (default = null) — null 以外の場合は、生成された AnimatorBase インスタンスの initialPosition プロパティを設定します。
 
zIndex:int (default = -1) — 親が SimpleButton の場合、この値は 0 から始まるオブジェクトの z 順序です。この順序に従って、アニメーションが適用されます。ボタンにオブジェクトが 1 つしかない場合、zIndex には 0 ではなく、-1 が設定されます。デフォルト値は -1 です。
 
placeholderName:String (default = null) — null 以外の場合は、instanceNameFactoryClass によって作成された DisplayObject で置き換えられるステージ上のインスタンスを指定します。デフォルト値は null です。
 
instanceFactoryClass:Class (default = null) — null 以外の場合は、プレースホルダーを置き換える DisplayObject の作成に使用するクラスです。デフォルト値は null です。

戻り値
AnimatorBase — 新しい AnimatorBase インスタンスです。
