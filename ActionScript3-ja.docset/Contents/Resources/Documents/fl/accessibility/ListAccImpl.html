ListAccImpl クラスは List Accessibility Implementation クラスとも呼ばれ、List コンポーネントにアクセシビリティ機能を適用します。

ListAccImpl クラスはシステムロール、オブジェクトベースのイベント、および状態をサポートします。

List はロール ROLE_SYSTEM_LIST（0x21）をスクリーンリーダーに報告します。List の各アイテムはロール ROLE_SYSTEM_LISTITEM（0x22）を報告します。

List は次の状態をスクリーンリーダーに報告します。

  • STATE_SYSTEM_NORMAL（0x00000000）
  • STATE_SYSTEM_UNAVAILABLE（0x00000001）
  • STATE_SYSTEM_FOCUSED（0x00000004）
  • STATE_SYSTEM_FOCUSABLE（0x00100000）

さらに、List の各アイテムは次の状態を報告します。

  • STATE_SYSTEM_SELECTED（0x00000002）
  • STATE_SYSTEM_FOCUSED（0x00000004）
  • STATE_SYSTEM_INVISIBLE（0x00008000）
  • STATE_SYSTEM_OFFSCREEN（0x00010000）
  • STATE_SYSTEM_SELECTABLE（0x00200000）

List は次のイベントをスクリーンリーダーに送り出します。

  • EVENT_OBJECT_FOCUS（0x8005）
  • EVENT_OBJECT_SELECTION（0x8006）
  • EVENT_OBJECT_STATECHANGE（0x800A）
  • EVENT_OBJECT_NAMECHANGE（0x800C）
