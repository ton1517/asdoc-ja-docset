クラスpublic class SliderEvent
継承SliderEvent Event Object

言語バージョン: ActionScript 3.0
製品バージョン: Flash CS3
ランタイムバージョン: Flash Player 9.0.28.0, AIR 1.0

SliderEvent クラスは、Slider コンポーネントに関連するイベントを定義します。次のような情報があります。
  • SliderEvent.CHANGE：スライダーの位置が変更されると、送出されます。
  • SliderEvent.THUMB_DRAG：ユーザーがスライダーのサムをドラッグしたときに送出されます。
  • SliderEvent.THUMB_PRESS：ユーザーがスライダーのサムを押したときに送出されます。
  • SliderEvent.THUMB_RELEASE：ユーザーがスライダーのサムを離したときに送出されます。

SliderEvent

()コンストラクター
public function SliderEvent(type:String, value:Number, clickTarget:String, triggerEvent:String, keyCode:int = 0)

言語バージョン: ActionScript 3.0
製品バージョン: Flash CS3
ランタイムバージョン: Flash Player 9.0.28.0, AIR 1.0

指定されたパラメーターで、新しい SliderEvent オブジェクトを作成します。

パラメーター
type:String — イベントタイプ。この値は、イベントをトリガーしたアクションを識別します。
 
value:Number — スライダーの新しい値です。
 
clickTarget:String — スライダーサムまたはスライダートラックが押されたかどうかを示します。値 SliderEventClickTarget.THUMB は、スライダーサムが押されたことを示します。値 SliderEventClickTarget.TRACK は、スライダートラックが押されたことを示します。
 
triggerEvent:String — 入力ソースを示すストリングです。値 InteractionInputType.MOUSE は、入力ソースがマウスであったことを示します。値 InteractionInputType.KEYBOARD は、入力ソースがキーボードであったことを示します。
 
keyCode:int (default = 0) — イベントがキーを押すことでトリガーされた場合、この値はそのキーを識別するキーコードです。

CHANGE

定数
public static const CHANGE:String = "change"

言語バージョン: ActionScript 3.0
製品バージョン: Flash CS3
ランタイムバージョン: Flash Player 9.0.28.0, AIR 1.0

type プロパティ（change イベントオブジェクト）の値を定義します。

このイベントには、次のプロパティがあります。

プロパティ値
bubblesfalse
cancelablefalse は、キャンセルするデフォルトの動作がないことを示します。
clickTargetスライダートラックまたはスライダーサムが押されたかどうかを識別します。
currentTargetイベントリスナーで event オブジェクトをアクティブに処理しているオブジェクトです。
keyCodeイベントがキーを押すことでトリガーされた場合は、押されたキーのコードです。
targetイベントを送出したオブジェクトです。ターゲットは、必ずしもイベントを受け取るオブジェクトではありません。イベントを受け取るオブジェクトにアクセスするには、currentTarget プロパティを使用します。
triggerEventイベントをトリガーしたデバイスのタイプです。値 InteractionInputType.MOUSE は、入力ソースがマウスであったことを示します。値 InteractionInputType.KEYBOARD は、入力ソースがキーボードであったことを示します。
valueイベント後のスライダーの値です。
