クラスpublic class ListEvent
継承ListEvent Event Object
サブクラス DataGridEvent

言語バージョン: ActionScript 3.0
製品バージョン: Flash CS3
ランタイムバージョン: Flash Player 9.0.28.0, AIR 1.0

ListEvent クラスは、List、DataGrid、TileList、ComboBox コンポーネントをはじめとしたリストコンポーネントのイベントを定義します。次のイベントがこれに該当します。
  • ListEvent.ITEM_CLICK：ユーザーがコンポーネント内のアイテムにマウスを合わせてクリックすると、送出されます。
  • ListEvent.ITEM_DOUBLE_CLICK：ユーザーがコンポーネント内のアイテムにマウスを合わせ、 2 回続けてすばやくクリックすると、送出されます。
  • ListEvent.ITEM_ROLL_OUT：ユーザーがマウスポインターをコンポーネントのアイテム外に移動すると、送出されます。
  • ListEvent.ITEM_ROLL_OVER：ユーザーがマウスポインターをコンポーネント内のアイテム上に重ねると、送出されます。

言語バージョン: ActionScript 3.0

ランタイムバージョン: Flash Player 9.0.28.0, AIR 1.0

指定されたパラメーターで、新しい ListEvent オブジェクトを作成します。

パラメーター
type:String — イベントタイプ。この値は、イベントの基となるアクションを識別します。
 
bubbles:Boolean (default = false) — イベントが表示リスト階層を上方にバブルできるかどうかを示します。
 
cancelable:Boolean (default = false) — イベントに関連付けられた動作を回避できるかどうかを示します。
 
columnIndex:int (default = -1) — レンダラーを含んでいる列の 0 から始まるインデックス、または列のデータの視覚的表現です。
 
rowIndex:int (default = -1) — レンダラーを含んでいる行の 0 から始まるインデックス、または行のデータの視覚的表現です。
 
index:int (default = -1) — DataProvider のアイテムの 0 から始まるインデックスです。
 
item:Object (default = null) — レンダラーに属するデータへの参照です。

例 この例の使用方法
ListEventExample.as

この例では、ListEvent クラスを使用してデータグリッド内のアイテムの行と列を判断する方法を示します。
  1. DataGrid コンポーネントをライブラリに追加します。
  2. このコードを FLA と同じディレクトリに ListEventExample.as という名前で保存します。
  3. FLA 内の DocumentClass を ListEventExample に設定します。
コピー

package
{
    import flash.display.Sprite;
    import fl.data.DataProvider;
    import fl.controls.DataGrid;
    import fl.events.ListEvent;
    
    public class ListEventExample extends Sprite
    {
        private var dg:DataGrid;
        
        public function ListEventExample() {
            createDataGrid();
        }
        private function createDataGrid():void {
            dg = new DataGrid();
            addChild(dg);
            dg.setSize(200,300);
            
            var dp:DataProvider = new DataProvider();
            var i:uint;
            var totalRows:uint = 42;
            for(i=0; i<totalRows; i++) {
                dp.addItem( {     col1:getRandomNumber(),
                                   col2:getRandomNumber(),
                                col3:getRandomNumber() } );
            }
            dg.dataProvider = dp;
            dg.addEventListener(ListEvent.ITEM_ROLL_OVER, declarePosition);
        }
        private function declarePosition(e:ListEvent):void {
            trace("You are above row: " + e.rowIndex + " column: " + e.columnIndex);    
        }
        private function getRandomNumber():uint {
            return Math.round(Math.random()*100);
        }
    }
}
