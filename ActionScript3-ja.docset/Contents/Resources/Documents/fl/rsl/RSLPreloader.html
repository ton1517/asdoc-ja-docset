クラスpublic class RSLPreloader
継承RSLPreloader EventDispatcher Object

言語バージョン: ActionScript 3.0
製品バージョン: Flash CS5
ランタイムバージョン: Flash Player 10.1, AIR 2

RSLPreloader クラスは、他のコンテンツを再生する前に、RSL（ランタイム共有ライブラリ）のプリロードを管理します。SWF（符号なし）ファイルと SWZ（符号付き、キャッシュ済み）ファイルの両方を処理します。RSLPreloader は、イベント（RSLEvent.RSL_LOAD_COMPLETE、RSLErrorEvent.RSL_LOAD_FAILED または RSLEvent.RSL_PROGRESS）を送出し、RSL ファイルのロードのステータスを示します。

プロパティの詳細

debugWaitTime

プロパティ
debugWaitTime:int

言語バージョン: ActionScript 3.0
製品バージョン: Flash CS5
ランタイムバージョン: Flash Player 10.1, AIR 2

最初の RSL ファイルをダウンロードするまで待機するミリ秒数の値を設定します。この方法で、エンドユーザーエクスペリエンスをシミュレートしたり、プリロードループをテストしたりできます。この方法は、SWZ キャッシュをフラッシュせずにロードの遅延時間を組み込んで符号付きキャッシュ RSL（SWZ ファイル）をテストする場合に特に有用です。この値を 0 より大きい値に設定するコードがある場合、それらのコードすべてをデプロイする前に無効にしてください。デバッグ構成定数（CONFIG::DEBUG など）を使用してコードを呼び出します。または、デバッグコードを忘れずに削除するために、5000（5 秒に相当）など、比較的大きな値に遅延時間を設定する方法もあります。オーサリングによって自動生成される RSLLoader インスタンスについては、アニメーション SWF のロード機能またはメインタイムラインコードで setRSLPreloader(value:RSLPreloader):void メソッドを定義することで遅延時間を設定できます。

コピー

         import fl.rsl.RSLPreloader;
         function setRSLPreloader(preloader:RSLPreloader):void {
         preloader.debugWaitTime = 10000;
         }

デフォルト値： 0。

         myPreloader.addRSLInfo(info);
         myPreloader.start();

            var urls:Array = myPreloader.getRSLInfoAt(i).rslURLs;

               trace(' url: ' + urls[j]);
            }
         }

イベントの詳細

rslLoadComplete

 イベント
イベントオブジェクトの型: fl.events.RSLEvent
プロパティ RSLEvent.type = fl.events.RSLEvent.RSL_LOAD_COMPLETE

言語バージョン: ActionScript 3.0
製品バージョン: Flash CS5
ランタイムバージョン: Flash Player 10.1, AIR 2

すべての RSL のロード完了時に、RSLPreloader によって送出されます。

すべての RSL のダウンロードが正常に完了したときに、RSLPreloader によって送出されるイベントです。このイベントタイプの bytesLoaded プロパティと bytesTotal プロパティは、常に 0 です。カスタムイベントループを使用してプリロードを実行している場合、ユーザーコンテンツはこのイベントをリッスンして、アニメーションループのロードを終了するタイミングを判定する必要があります。

このイベントには、次のプロパティがあります。

プロパティ値
bubblestrue
cancelablefalse は、キャンセルするデフォルトの動作がないことを示します。
currentTargetイベントリスナーで Event オブジェクトをアクティブに処理しているオブジェクトです。
target表示リストに追加される DisplayObejct インスタンスです。target は、必ずしもイベントリスナーを登録した表示リスト内のオブジェクトとは限りません。現在イベントを処理している表示リスト内のオブジェクトにアクセスするには、currentTarget プロパティを使用します。
rslsLoadedダウンロードに成功したファイル数です。
rslsFailedダウンロードに失敗したファイル数です。
rslsTotalダウンロードしたファイルの総数（成功および失敗を含む）です。
bytesLoadedダウンロードに成功したファイルについて、ダウンロードされたバイトの総数です。
bytesTotalダウンロードしたファイルについて、ダウンロードされたバイトの総数（成功および失敗を含む）です。
コピー

         import fl.events.RSLEvent;
         myPreloader.addEventListener(RSLEvent.RSL_LOAD_COMPLETE, loadComplete);
         function loadComplete(e:RSLEvent) {
         gotoAndPlay('startOfContent');
         }

すべての RSL のダウンロードが終了したとき、失敗したファイルが 1 つ以上ある場合に、RSLPreloader によって送出されます。

すべての RSL のダウンロード終了時、失敗したファイルが 1 つ以上ある場合に、RSLPreloader によって送出されるエラーです。カスタムイベントループを使用してプリロードを実行している場合、ユーザーコンテンツはこのイベントをリッスンして、アニメーションループのロードを終了するタイミングを判定する必要があります。

このイベントには、次のプロパティがあります。

プロパティ値
bubblestrue
cancelablefalse は、キャンセルするデフォルトの動作がないことを示します。
currentTargetイベントリスナーで Event オブジェクトをアクティブに処理しているオブジェクトです。
target表示リストに追加される DisplayObejct インスタンスです。target は、必ずしもイベントリスナーを登録した表示リスト内のオブジェクトとは限りません。現在イベントを処理している表示リスト内のオブジェクトにアクセスするには、currentTarget プロパティを使用します。
rslsLoadedダウンロードに成功したファイル数です。
rslsFailedダウンロードに失敗したファイル数です。
rslsTotalダウンロードしたファイルの総数（成功および失敗を含む）です。
failedURLsダウンロードに失敗したファイルの配列です。
コピー

         import fl.events.RSLErrorEvent;
         myPreloader.addEventListener(RSLErrorEvent.RSL_LOAD_FAILED, rslLoadFailedHandler);
         function rslLoadFailedHandler(e:RSLErrorEvent) {
         gotoAndPlay('rslLoadFailure');
         }

rslProgress

 イベント  
イベントオブジェクトの型: fl.events.RSLEvent
プロパティ RSLEvent.type = fl.events.RSLEvent.RSL_PROGRESS

言語バージョン: ActionScript 3.0
製品バージョン: Flash CS5
ランタイムバージョン: Flash Player 10.1, AIR 2

RSLPreloader によって送出され、RSL ファイルのダウンロードの進捗状況を示します。

RSLPreloader によって送出されるイベントで、RSL ファイルのダウンロードの進捗状況を示します。このイベントタイプは、bytesLoaded プロパティおよび bytesTotal プロパティがゼロ以外の場合にのみ送出されます。

このイベントには、次のプロパティがあります。

プロパティ値
bubblestrue
cancelablefalse は、キャンセルするデフォルトの動作がないことを示します。
currentTargetイベントリスナーで Event オブジェクトをアクティブに処理しているオブジェクトです。
target表示リストに追加される DisplayObejct インスタンスです。target は、必ずしもイベントリスナーを登録した表示リスト内のオブジェクトとは限りません。現在イベントを処理している表示リスト内のオブジェクトにアクセスするには、currentTarget プロパティを使用します。
rslsLoadedダウンロードに成功したファイル数です。
rslsFailedダウンロードに失敗したファイル数です。
rslsTotalダウンロードしたファイルの総数（成功および失敗を含む）です。
bytesLoadedダウンロードに成功したファイルについて、ダウンロードされたバイトの総数です。
bytesTotalダウンロードしたファイルについて、ダウンロードされたバイトの総数（成功および失敗を含む）です。
コピー

         import fl.events.RSLEvent;
         myPreloader.addEventListener(RSLEvent.RSL_PROGRESS, progressHandler);
         function progressHandler(e:RSLEvent) {
         trace('rslsLoaded = ' + e.rslsLoaded);
         trace('rslsFailed = ' + e.rslsFailed);
         trace('rslsTotal = ' + e.rslsTotal);
         trace('bytesLoaded = ' + e.bytesLoaded);
         trace('bytesTotal = ' + e.bytesTotal);
         }
