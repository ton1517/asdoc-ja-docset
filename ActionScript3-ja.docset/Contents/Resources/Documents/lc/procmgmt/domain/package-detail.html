lc.procmgmt.domain パッケージには、プロセス管理の概念（ビジネスドメインとも呼ばれます）をカプセル化するクラスが含まれます。このパッケージのクラスは、Document Server および Adobe Digital Enterprise Platform Document Services - Workspace のデータへのアクセスを必要とする、Flex を使用して構築されたアプリケーションに役立ちます。呼び出された操作に対して、Document Server 上で追加の操作が発生する可能性があります。
