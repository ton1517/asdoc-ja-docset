content

プロパティ 
content:*

言語バージョン: ActionScript 3.0
ランタイムバージョン: Flash Player 10, AIR 1.5

マーカーのコンテンツを制御します。

このストリングの有効な値は次のとおりです。

  • none - マーカーなし。
  • counter(ordered) - マーカーを表示します。
  • counter(ordered,ListStyleType) - マーカーを表示します。ただし、listStyleType を指定された値に変更します。
  • counters(ordered) - 最上位の親 ListElement から開始して、順序付けされたカウンターの値のストリングを作成します。各カウンターの指定の listStyleType はそれぞれ接尾辞で区切られます。これはアウトライン番号に使用されます（例えば、I.1.、I.2. など）。
  • counters(ordered,"<string>") - 順序付けされた各カウンターの接尾辞が <string> に置き換えられることを除き、前の値と同じです。
  • counters(ordered,"<string>",ListStyleType) - 各カウンターの listStyleType が指定の値に置き換えられる点を除き、前の値と同じです。

定義しない場合、このプロパティのデフォルト値は "counter(ordered)" です。



実装
    public function get content():*
    public function set content(value:any):void

例外
RangeError — 設定値がこのプロパティの範囲にない場合。

counterIncrement

プロパティ 
counterIncrement:*

言語バージョン: ActionScript 3.0
ランタイムバージョン: Flash Player 10, AIR 1.5

カウンターの値の増分を制御します。

このストリングの有効な値は次のとおりです。

  • none - 増分なし。
  • ordered - カウンターを 1 ずつ増分します。
  • ordered integer - カウンターを integer ずつ増分します。

定義しない場合、このプロパティのデフォルト値は "ordered 1" です。

注意：counterIncrement プロパティは、counterReset プロパティの前に適用されます。



実装
    public function get counterIncrement():*
    public function set counterIncrement(value:any):void

例外
RangeError — 設定値がこのプロパティの範囲にない場合。

counterReset

プロパティ 
counterReset:*

言語バージョン: ActionScript 3.0
ランタイムバージョン: Flash Player 10, AIR 1.5

カウンターの値のリセットを制御します。

このプロパティの有効な値は次のとおりです。

  • none - リセットなし。
  • ordered - カウンターを 0 にリセットします。
  • ordered integer - カウンターを integer にリセットします。

定義しない場合、このプロパティのデフォルト値は「none」です。

注意：counterReset プロパティは、counterIncrement プロパティの前に適用されます。

suffix

プロパティ 
suffix:*

言語バージョン: ActionScript 3.0
ランタイムバージョン: Flash Player 10, AIR 1.5

ListItemElement 内の生成されたテキストへの接尾辞の適用を制御します。

有効な値は次のとおりです。

  • flashx.textLayout.formats.Suffix.NONE - 接尾辞なし。

デフォルト値は Suffix.AUTO です。
