クラスpublic class TextRange
継承TextRange Object
サブクラス SelectionState

言語バージョン: ActionScript 3.0
ランタイムバージョン: Flash Player 10, AIR 1.5

隣接するテキストの範囲を表す読み取り専用クラスです。テキストのセクションを選択した場合に、このような範囲が発生します。範囲は、選択範囲のアンカーポイント、anchorPosition および activePosition（アクションにより変更されるポイント）により構成されます。ブロックの選択範囲が変更および拡張されると、anchorPosition は固定されたままで、activePosition は変更されます。アンカー位置は、アクティブ位置の前または後のテキスト内に配置される場合があります。

TextRange

()コンストラクター
public function TextRange(root:flashx.textLayout.elements:TextFlow, anchorIndex:int, activeIndex:int)

言語バージョン: ActionScript 3.0
ランタイムバージョン: Flash Player 10, AIR 1.5

コンストラクター - 新しい TextRange インスタンスを作成します。TextRange は範囲なしを意味する (-1,-1)、または 0 から TextFlow.textLength までの対の値を指定できます。

パラメーター
root:flashx.textLayout.elements:TextFlow — 選択範囲に関連付けられている TextFlow です。
 
anchorIndex:int — 選択範囲内のアンカーのインデックス位置です。テキスト内の最初の位置は、位置 0 となります。
 
activeIndex:int — 選択範囲内のアクティブ位置のインデックス位置です。テキスト内の最初の位置は、位置 0 となります。

関連する API エレメント

戻り値
