flashx.textLayout.elements パッケージには、テキストコンテンツを Text Layout Framework で表現するためのコアクラスが含まれます。


インターフェイス
 インターフェイス説明
 IConfiguration 構成オブジェクトの読み取り専用インターフェイスです。
 IFormatResolver フォーマットリゾルバーへのインターフェイスです。
クラス
 クラス説明
 BreakElement BreakElement クラスは、改行を定義して、新しい段落を作成せずにテキスト内で改行を作成できるようにします。
 Configuration Configuration クラスは、Text Layout Framework とアプリケーション間の統合のプライマリーポイントです。
 ContainerFormattedElement ContainerFormattedElement は、DivElement オブジェクトや TextFlow オブジェクトなどのすべてのコンテナレベルのブロックエレメントのルートクラスです。
 DivElement DivElement クラスは、段落（ParagraphElement オブジェクト）をグループ化するエレメントを定義します。
 FlowElement フロー内のテキストは、テキスト内の論理的分割を表すツリーのエレメントとともにツリーフォームに保存されます。
 FlowGroupElement FlowGroupElement クラスは、子の配列を持つことができる FlowElement オブジェクトの基本クラスです。
 FlowLeafElement フロー階層の最下位に表示される FlowElements の基本クラスです。
 GlobalSettings すべての TextFlow オブジェクトに適用される設定です。
 InlineGraphicElement InlineGraphicElement クラスは、テキスト内にインラインを表示するグラフィックオブジェクトを処理します。
 InlineGraphicElementStatus InlineGraphicElementStatus クラスは、InLineGraphicElement.status の値をチェックするための一連の定数を定義します。
 LinkElement LinkElement クラスは、URI（Universal Resource Identifier）へのリンクを定義します。これは、ユーザーがリンクをクリックすると実行されます。
 LinkState LinkState クラスは、LinkElement クラスの linkState プロパティの一連の定数を定義します。
 ListElement List クラスは、アイテムを番号付きリストまたは番号なしリストにグループ化するために使用されます。
 ListItemElement ListItemElement はリスト内のアイテムです。
 OverflowPolicy OverflowPolicy クラスは、IConfiguration クラスの overflowPolicy プロパティの一連の定数を定義します。
 ParagraphElement ParagraphElement クラスは、テキストフロー階層で段落を表します。
 ParagraphFormattedElement ParagraphFormattedElement クラスは、段落プロパティを持つ FlowElement クラスの抽象基本クラスです。
 SpanElement SpanElement クラスは、適用される単一のフォーマット属性セットを含むテキストの実行を表します。
 SpecialCharacterElement SpecialCharacterElement クラスは、特殊文字を表すエレメントの抽象基本クラスです。
 SubParagraphGroupElement SubParagraphGroupElement は、FlowLeafElements、および SubParagraphGroupElementBase を拡張するその他のクラスをグループ化するエレメントです。
 SubParagraphGroupElementBase SubParagraphGroupElementBase クラスは、複数の FlowLeafElement をグループ化します。
 TabElement TabElement クラスは、テキストフロー内の <tab/> を表します。
 TCYElement TCYElement（Tatechuuyoko - 縦中横）クラスは、SubParagraphGroupElementBase のサブクラスであり、縦書きの行の中でテキストを横書きさせます。
 TextFlow TextFlow クラスは、ストーリーのすべてのテキストコンテンツを管理する役割を果たします。
 TextRange 隣接するテキストの範囲を表す読み取り専用クラスです。

[ X ]英語で表示される理由
ActionScript 3.0 リファレンスガイドのコンテンツが英語で表示されます。

ActionScript 3.0 リファレンスガイドのすべての部分がすべての言語に翻訳されているわけではありません。言語エレメントが翻訳されていない場合、そのエレメントは英語で表示されます。例えば、ga.controls.HelpBox クラスはどの言語にも訳されていません。このため、リファレンスガイドの日本語バージョンでは、ga.controls.HelpBox クラスは英語で表示されます。
