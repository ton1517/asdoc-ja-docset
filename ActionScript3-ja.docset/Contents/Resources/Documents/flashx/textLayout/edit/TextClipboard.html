クラスpublic class TextClipboard
継承TextClipboard Object

言語バージョン: ActionScript 3.0
ランタイムバージョン: Flash Player 10, AIR 1.5

TextClipboard クラスは、システムクリップボードとの間で TextScrap オブジェクトをコピーおよびペーストします。

TextScrap を TextClipboard にコピーする際、情報は 2 つのクリップボードフォーマットでシステムクリップボードにコピーされます。1 つめは XML ストリングで、コピーした TextScrap オブジェクトを Text Layout Markup 構文で表します。clipboard オブジェクトはフォーマット名 "TEXT_LAYOUT_MARKUP" を使用します。2 つめはプレーンテキストストリングで、標準の Clipboard.TEXT_FORMAT 名を使用します。

TextClipboard クラスのメソッドは静的関数であるため、TextClipboard のインスタンスを作成する必要はありません。

getContents

()メソッド
public static function getContents():flashx.textLayout.edit:TextScrap

言語バージョン: ActionScript 3.0
ランタイムバージョン: Flash Player 10, AIR 1.5

システムクリップボードのテキストを TextScrap オブジェクトとして取得します。

"TEXT_LAYOUT_MARKUP" フォーマットを使用できる場合、このメソッドはフォーマットしたストリングを TextScrap に変換して返します。このフォーマットを使用できず、Clipboard.TEXT_Format を使用できる場合、このメソッドはプレーンテキストストリングを TextScrap に変換します。どちらのクリップボードフォーマットも使用できない場合、このメソッドは null を返します。

Flash Player では、getContents() メソッドをペーストイベントハンドラーで呼び出す必要があります。AIR では、この制限はアプリケーションセキュリティサンドボックス外のコンテンツにのみ適用されます。

setContents

()メソッド 
public static function setContents(textScrap:flashx.textLayout.edit:TextScrap):void

言語バージョン: ActionScript 3.0
ランタイムバージョン: Flash Player 10, AIR 1.5

TextScrap をシステムクリップボードに置きます。

TextScrap は Text Layout Markup 表現およびプレーンテキスト表現の両方で、システムクリップボードに置かれます。

Flash Player では、setContents() を呼び出す前に、キー入力やマウスのクリックなどのユーザーイベントを必要とします。AIR では、この制限はアプリケーションセキュリティサンドボックス外のコンテンツにのみ適用されます。
