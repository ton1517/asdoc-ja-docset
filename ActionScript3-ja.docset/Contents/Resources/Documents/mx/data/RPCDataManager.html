updateItem() の呼び出しで競合検出を行えるようにするには、このプロパティを「object」または「property」に設定します。「object」モードの場合、最初に使用したバージョンのオブジェクトと競合するサーバーバージョンのオブジェクトに対して加えられた変更はすべて、競合と見なされます。「property」モードでは、更新しようとしているプロパティが、編集を開始した後に異なる値に変更された場合にのみ、競合が発生したと見なされます。このモードでは、オブジェクトを削除できます。

conflict

 イベント
イベントオブジェクトの型: mx.data.events.DataConflictEvent
プロパティ DataConflictEvent.type = mx.data.events.DataConflictEvent.CONFLICT

言語バージョン: ActionScript 3.0
製品バージョン: Adobe Digital Enterprise Platform Data Services for Java EE 3
ランタイムバージョン: Flash Player 9, AIR 1.1

ローカルで保留状態の変更と別のクライアントによって送信された変更の間で競合が検出された場合、またはこのクライアントによって送信された変更がリモート宛先での変更と競合している場合は、DataConflictEvent.CONFLICT イベントが送出されます。各データマネージャーは、その宛先のアイテムに対してのみ、競合イベントを送出します。関連する複数の宛先の競合を一度に処理する必要がある場合は、DataService で競合イベントを使用します。

CONFLICT 定数により、conflict イベントのイベントオブジェクトの type プロパティ値が定義されます。

イベントオブジェクトの各プロパティには次の値が設定されています。

プロパティ値
bubblesfalse
cancelablefalse
currentTargetイベント処理を行うイベントリスナーの定義用オブジェクトを表します。 例えば、イベントリスナーの登録に myButton.addEventListener() を使用すると、myButton の値は currentTarget となります。
conflict 発生した競合を保持する Conflict オブジェクトです。
message このイベントに関連付けられたメッセージ。
targetイベントを送出したオブジェクトを表します。このオブジェクトは必ずしもイベントを受け取るオブジェクトとは限りません。 イベントを受け取るオブジェクトに常にアクセスするには、currentTarget プロパティを使用してください。

fault

 イベント  
イベントオブジェクトの型: mx.data.events.DataServiceFaultEvent
プロパティ DataServiceFaultEvent.type = mx.data.events.DataServiceFaultEvent.FAULT

言語バージョン: ActionScript 3.0
製品バージョン: Adobe Digital Enterprise Platform Data Services for Java EE 3
ランタイムバージョン: Flash Player 9, AIR 1.1

エラーが発生してサービスの呼び出しに失敗すると、DataServiceFaultEvent.FAULT イベントが送出されます。例えば、このイベントは、fill() メソッドの呼び出しに失敗した場合などに送出されます。また、このイベントは、コミット結果の一部としてエラーを生成する各アイテムに関しても送出されます。その場合は、コミットによって返されたトークンが DataServiceFaultEvent に含まれていますが、フォルト内のアイテムおよびエラーメッセージのストリングは、エラーの原因になったアイテムごとに異なります。

FAULT 定数により、fault イベントのイベントオブジェクトの type プロパティ値が定義されます。

イベントオブジェクトの各プロパティには次の値が設定されています。

プロパティ値
bubblesfalse
cancelabletrue。関連付けられたトークンの responder.fault メソッドから preventDefault() を呼び出すと、サービスまたは処理がこのイベントを送出しなくなります。
currentTargetイベント処理を行うイベントリスナーの定義用オブジェクトを表します。 例えば、イベントリスナーの登録に myButton.addEventListener() を使用すると、myButton の値は currentTarget となります。
fault 発生した競合を保持する Fault オブジェクトです。
item フォルトが発生したアイテムです。
identity フォルトが発生したアイテムの ID です。
message このイベントに関連付けられたメッセージ。
token メソッドの呼び出しを表すトークン。Asynchronous Completion Token パターンで使用されます。
targetイベントを送出したオブジェクトを表します。このオブジェクトは必ずしもイベントを受け取るオブジェクトとは限りません。 イベントを受け取るオブジェクトに常にアクセスするには、currentTarget プロパティを使用してください。

 イベント

プロパティ PropertyChangeEvent.type = mx.events.PropertyChangeEvent.PROPERTY_CHANGE

言語バージョン: ActionScript 3.0
製品バージョン: Adobe Digital Enterprise Platform Data Services for Java EE 3

このサービスのプロパティが変更されると、PropertyChangeEvent.PROPERTY_CHANGE イベントが送出されます。この動作は次のプロパティに関して発生します。
