ManagedOperation

()コンストラクター
public function ManagedOperation(nameParam:String = null, typeParam:String = null)

言語バージョン: ActionScript 3.0
製品バージョン: Adobe Digital Enterprise Platform Data Services for Java EE 3
ランタイムバージョン: Flash Player 9, AIR 1.1

ManagedOperation オブジェクトを作成します。name と type は両方とも必須のパラメーターですが、ここではこのクラスを MXML タグとして使用できるようにオプションにしています。

パラメーター
nameParam:String (default = null) — 現在これによって管理されているサービス内の操作の名前です。
 
typeParam:String (default = null) — 管理対象操作のタイプです。値は、「create」、「update」、「delete」、「get」、「query」、または「findItem」です。
