ManagedObjectProxy クラスは、DataService によって管理される匿名アイテムに対する変更を追跡する機能を提供します。addEventListener() メソッドを使用することによって、任意の数のオブジェクトが、このオブジェクトでの変更を「リスニング」できます。このクラスは、[Managed] メタデータが適用されているクラスに与えられるものと同じレベルの機能を、DataService によって管理される匿名オブジェクトに提供します。

指定したオブジェクトと ID を使用してこのプロキシを初期化します。

パラメーター
item:Object (default = null) — プロキシするオブジェクトです。アイテムが指定されていない場合は、匿名オブジェクトが作成されて割り当てられます。
 
uid:String (default = null) — このオブジェクトインスタンスの一意の ID を含むストリングです。すべてのオブジェクトは一意に識別できる必要があるため、IManaged に準拠するために必要です。値が指定されていない場合は、任意の ID が割り当てられます。

DataService は、含まれているアイテムのいずれかに対する変更が修正される前に、非管理対象コレクションの状態を保存する必要があります。これを行うには、リストの実装からの PropertyChangeEvents をリスニングし、DataService がイベントをインターセプトしてコレクションの適切なスナップショットを作成できるように、イベントを再送出する必要があります。非管理対象コレクションを検出するには、リストの実装を調べます。管理対象コレクションは DataList を使用し、非管理対象コレクションは IPropertyChangeNotifier を実装する必要があるリストを使用します。

パラメーター

name:* — 通常は、プロパティの名前を含むストリングです。localName プロパティの検証でプロパティ名が検出された場合は、QName になります。

戻り値
* — ObjectProxy のプロパティ値またはインスタンスです。

DataService の宛先が定義されていて、遅延関連付けが設定されている場合、ManagedObjectProxy はラップされた匿名オブジェクトでプロパティを除外することが必要になる場合があります。このメソッドは、writeExternal のカスタム直列化をサポートします。

パラメーター

input:IDataInput — 直列化されたデータのソースです。
