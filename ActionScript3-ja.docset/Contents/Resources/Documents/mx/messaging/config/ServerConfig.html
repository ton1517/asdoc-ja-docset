このクラスは、サーバーのメッセージング設定情報へのアクセスを提供します。 このクラスはクライアントの services-config.xml ファイルからの情報をカプセル化します。このクラスは、メッセージングシステムが設定済みの ChannelSet および Channel をメッセージングフレームワークに提供する際に使用されます。

コンパイル時には XML ソースが提供されます。 ただし現在、ネットワーク、ローカルファイルシステム、実行時の共有オブジェクトなどという他の方法によるこの XML データの取得を防止する内部制約はありません。

checkChannelConsistency

()メソッド
public static function checkChannelConsistency(destinationA:String, destinationB:String):void

言語バージョン: ActionScript 3.0
製品バージョン: BlazeDS 4, Adobe Digital Enterprise Platform Data Services for Java EE 3
ランタイムバージョン: Flash Player 9, AIR 1.1

このメソッドは、指定された 2 つの宛先が、同一のチャンネル定義を持つことを確認します。 指定された 2 つの宛先のチャンネル定義が同一でない場合、このメソッドは ArgumentError をスローします。

パラメーター

destinationA:String — 比較する最初の宛先です。
 
destinationB:String — チャンネルを比較する 2 番目の宛先です。

言語バージョン: ActionScript 3.0
製品バージョン: BlazeDS 4, Adobe Digital Enterprise Platform Data Services for Java EE 3
ランタイムバージョン: Flash Player 9, AIR 1.1

設定された Channel の共有インスタンスを返します。

パラメーター

id:String — 目的の Channel の id です。
 
clustered:Boolean (default = false) — Channel がクラスタ化されて使用される場合は true、そうでない場合は false。

戻り値
Channel — Channel インスタンスです。

例外
InvalidChannelError — 指定された id を持つ Channel がない場合。
