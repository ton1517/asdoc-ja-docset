メッセージの相関 ID へのアクセスを提供します。 メッセージの応答と分割に使用されます。 correlationId には、このメッセージが参照する前のメッセージの messageId が含まれます。

言語バージョン: ActionScript 3.0

本文とヘッダーが空の AsyncMessage インスタンスを作成します。 このデフォルト動作に加え、便宜上の目的でメッセージの本文とヘッダーをコンストラクターに渡すこともできます。 この本文呼び出し方法の例は var msg:AsyncMessage = new AsyncMessage("Body text"); です。本文とヘッダーの両方を提供する例は var msg:AsyncMessage = new AsyncMessage("Body text", {"customerHeader":"customValue"}); です。

パラメーター
body:Object (default = null) — メッセージに割り当てるオプションの本文です。
 
headers:Object (default = null) — メッセージに割り当てるオプションのヘッダーです。
