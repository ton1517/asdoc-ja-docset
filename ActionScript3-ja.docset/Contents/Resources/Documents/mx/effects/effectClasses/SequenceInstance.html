クラスpublic class SequenceInstance
継承SequenceInstance CompositeEffectInstance EffectInstance EventDispatcher Object

言語バージョン: ActionScript 3.0
製品バージョン: Flex 3
ランタイムバージョン: Flash Player 9, AIR 1.1

SequenceInstance クラスは、Sequence エフェクトのインスタンスクラスを実装します。 このクラスのインスタンスは、Sequence エフェクトの再生時に Flex によって作成されます。ユーザー自身が作成することはありません。

onEffectEnd

()メソッド 
override protected function onEffectEnd(childEffect:IEffectInstance):void

言語バージョン: ActionScript 3.0
製品バージョン: Flex 3
ランタイムバージョン: Flash Player 9, AIR 1.1

SequenceInstance の子エフェクトが終了するたびに、onEffectEnd() メソッドが呼び出されます。 SequenceInstance の場合、次のエフェクトが再生されます。 このメソッドは、スーパークラスのメソッドを実装します。

パラメーター

childEffect:IEffectInstance — 子エフェクトを表します。
