mx.effects.easing パッケージには、エフェクトで使用するイージングクラスが含まれています。


クラス
 クラス説明
 Back Back クラスでは、Flex エフェクトクラスでモーションを実装する 3 種類のイージング関数を定義します。
 Bounce Bounce クラスでは、Flex エフェクトクラスでバウンスモーションを実装する 3 種類のイージング関数を定義します。
 Circular Circular クラスでは、Flex エフェクトクラスで円形モーションを実装する 3 種類のイージング関数を定義します。
 Cubic Cubic クラスでは、Flex エフェクトクラスでモーションを実装する 3 種類のイージング関数を定義します。
 Elastic Elastc クラスでは、モーションが指数関数的に減衰する正弦波により定義される Flex エフェクトクラスでモーションを実装する 3 種類のイージング関数を定義します。
 Exponential Exponential クラスでは、モーションが指数関数的に減衰する正弦波により定義される Flex エフェクトクラスでモーションを実装する 3 種類のイージング関数を定義します。
 Linear Linear クラスでは、Flex エフェクトクラスで線形モーションを実装するイージング関数を定義します。
 Quadratic Quadratic クラスでは、Flex エフェクトクラスで二次モーションを実装する 3 種類のイージング関数を定義します。
 Quartic Quartic クラスでは、Flex エフェクトクラスでモーションを実装する 3 種類のイージング関数を定義します。
 Quintic Quintic クラスでは、Flex エフェクトクラスでモーションを実装する 3 種類のイージング関数を定義します。
 Sine Sine クラスでは、モーションが正弦波により定義される Flex エフェクトクラスでモーションを実装する 3 種類のイージング関数を定義します。
