RSLData

()コンストラクター
public function RSLData(rslURL:String = null, policyFileURL:String = null, digest:String = null, hashType:String = null, isSigned:Boolean = false, verifyDigest:Boolean = false, applicationDomainTarget:String = "default")

言語バージョン: ActionScript 3.0
製品バージョン: Flex 4.5
ランタイムバージョン: Flash Player 10.2, AIR 2.5

コンストラクターです。

パラメーター
rslURL:String (default = null) — RSL の場所です。
 
policyFileURL:String (default = null) — ポリシーファイルの url の場所です（オプション）。
 
digest:String (default = null) — RSL のダイジェストです。 ダイジェストがない RSL の場合は null です。
 
hashType:String (default = null) — ダイジェストを作成するために使用されたハッシュのタイプです。 サポートされている値は SHA256.TYPE_ID のみです。
 
isSigned:Boolean (default = false) — RSL が、アドビによって署名されている場合は true、それ以外の場合は false です。
 
verifyDigest:Boolean (default = false) — RSL のダイジェストを読み込んだ後に検証するかどうかを決定します。
 
applicationDomainTarget:String (default = "default") — RSL が読み込まれるアプリケーションドメインです。 有効な値については、ApplicationDomainTarget の列挙を参照してください。
