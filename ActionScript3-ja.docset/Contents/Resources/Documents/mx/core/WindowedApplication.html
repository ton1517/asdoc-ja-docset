Flex 4.0 以降では、このクラスの代わりとして spark.components.WindowedApplication クラスを使用することをお勧めします。

WindowedApplication は、AIR アプリケーション用の Flex アプリケーションを作成するために使用するアプリケーションコンテナを定義します。

WindowedApplication は、2 つの役割を果たします。これは <mx:Application> タグの代わりに使用され、Flex ベースの AIR アプリケーションへのエントリポイントとして機能します。また、WindowedApplication はコンテナとして、Flex AIR アプリケーションの最初のウィンドウのレイアウトを定義します。WindowedApplication で定義されたすべてのビジュアルコントロールは、AIR アプリケーションによってロードされる最初のウィンドウのコンテンツになります。

WindowedApplication は、Flex ベースの AIR アプリケーションの最上位アプリケーションです。他の Flex アプリケーションによってロードされるように設計されていません。WindowedApplication で他のアプリケーションをロードする必要がある場合は、これらのアプリケーションのコンテナとして Application クラスを使用します。他のウィンドウが必要な場合は、これらのウィンドウのコンテナとして Window クラスを使用します。

WindowedApplication は、マルチウィンドウアプリケーションのすべてのウィンドウではなく、単一のウィンドウのビジュアルコンテンツのみを表すため、WindowedApplication のインスタンスは、アプリケーション内の他のウィンドウで発生したイベントについてではなく、それ自身のステージおよびウィンドウについての表示関連のイベント（WindowedApplication クラスが InteractiveObject や UIComponent などの表示オブジェクトの基本クラスから継承したイベント）のみを送出します。この点で、Application コンテナがアプリケーションのすべてのウィンドウのイベントを送出する、ブラウザーベースのアプリケーションとは異なっています（ブラウザーベースのアプリケーションでは、技術的にはウィンドウはすべて、単一の Application ステージにレンダリングされる表示オブジェクトです）。

useNativeDragManager

プロパティ 
public var useNativeDragManager:Boolean = true

言語バージョン: ActionScript 3.0
製品バージョン: Flex 4
ランタイムバージョン: Flash Player 10, AIR 1.5

true の場合、DragManager は NativeDragManagerImpl 実装クラスを使用します。false の場合は、DragManagerImpl クラスを使用します。

メモ：このプロパティは ActionScript コードでは設定できません。MXML コードで設定する必要があります。つまり、実行時にその値は変更できません。

デフォルトでは、Flex で作成する AIR アプリケーションの DragManager は、実装クラスとして NativeDragManagerImpl クラスを使用します。Flex で作成する Flash Player アプリケーションは、DragManagerImpl クラスを使用します。

NativeDragManagerImpl クラスは、AIR NativeDragManager API および Flex DragManager API 間のブリッジです。AIR NativeDragManager クラスは、オペレーティングシステムのドラッグ＆ドロップ API を使用します。AIR ウィンドウ間のドラッグ、およびオペレーティングシステムと AIR 間のドラッグをサポートしています。オペレーティングシステムがドラッグ＆ドロップ操作を制御しているので、ドラッグ時のカーソルをカスタマイズすることはできません。また、ドロップアニメーションを制御することもできません。この動作はオペレーティングシステムに依存しており、異なるプラットフォームでは一貫性がない場合があります。

DragManagerImpl クラスは、ドラッグ＆ドロップ時にオペレーティングシステムを使用しません。代わりに、ドラッグ＆ドロッププロセス全体を制御します。このクラスは、カーソルのカスタマイズをサポートしており、ドロップアニメーションを提供します。ただし、AIR ウィンドウ間のドラッグ、およびオペレーティングシステムと AIR ウィンドウ間のドラッグは行うことができません。

デフォルト値： true。
