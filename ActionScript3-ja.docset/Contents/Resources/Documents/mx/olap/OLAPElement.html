OLAPElement クラスは、すべての OLAP エレメントに共通のプロパティを提供するベースインターフェイスを定義します。

MXML シンタックスMXML シンタックスを表示
MXML シンタックスを隠す

The <mx:OLAPElement> tag inherits all of the tag attributes of its superclass, and adds the following tag attributes: 


  <mx:OLAPElement
    Properties
    dimensions=""
    name=""
  />

name

name:String

言語バージョン: ActionScript 3.0
製品バージョン: Flex 3
ランタイムバージョン: Flash Player 9, AIR 1.1

エレメントの OLAP スキーマの階層を含む OLAP エレメントの名前です。 例えば、OLAP エレメントの名前に "Time_Year" があります。この場合、"Year" は OLAP スキーマの "Time" 次元のレベルです。

    public function set name(value:String):void

uniqueName

プロパティ 
uniqueName:String  [読み取り専用]

言語バージョン: ActionScript 3.0
製品バージョン: Flex 3
ランタイムバージョン: Flash Player 9, AIR 1.1

キューブ内の OLAP エレメントの一意の名前です。 例えば、"[Time][Year][2007]" は一意の名前です。この場合、"2007" は "Time" 次元の "Year" レベルに属するエレメントの名前です。
