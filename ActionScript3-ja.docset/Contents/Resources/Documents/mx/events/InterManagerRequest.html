コンストラクターの詳細

InterManagerRequest

()コンストラクター
public function InterManagerRequest(type:String, bubbles:Boolean = false, cancelable:Boolean = false, name:String = null, value:Object = null)

言語バージョン: ActionScript 3.0
製品バージョン: Flex 3
ランタイムバージョン: Flash Player 9, AIR 1.1

コンストラクターです。何も返しませんが、value プロパティはメソッドの戻り値を表すように変更できます。

パラメーター
type:String — イベントタイプ。イベントの基となるアクションを示します。
 
bubbles:Boolean (default = false) — イベントが表示リスト階層を上方にバブルできるかどうかを指定します。
 
cancelable:Boolean (default = false) — イベントに関連付けられた動作をキャンセルできるかどうかを指定します。
 
name:String (default = null) — プロパティ名、メソッド名、またはインスタンス化するマネージャーの名前です。
 
value:Object (default = null) — プロパティの値、またはメソッドのパラメーターの配列 (null でない場合) です。

INIT_MANAGER_REQUEST

定数

言語バージョン: ActionScript 3.0

何らかのアクションを実行するよう SystemManager に要求します。name プロパティは、実行するアクションの名前です。value プロパティは、そのアクションの実行に必要な値です。
