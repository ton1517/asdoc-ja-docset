クラスpublic class EventListenerRequest
継承EventListenerRequest SWFBridgeRequest Event Object

言語バージョン: ActionScript 3.0
製品バージョン: Flex 3
ランタイムバージョン: Flash Player 9, AIR 1.1

指定されたイベントへのリスナーを追加または削除するために、ある SystemManager からブリッジを通じて別のアプリケーションの SystemManager に送信された要求です。data プロパティは使用されません。特定のイベントのみ要求できます。イベントが他のアプリケーションでトリガーされた場合、イベントはブリッジを通じて再送出されます。ブリッジでは、要求元の SystemManager がイベントを選択して自身から再送出します。一般に、この要求が生成されるのは、他のコードが SystemManager での承認済みイベントの 1 つに対して addEventListener() メソッドを呼び出したためです。また、この要求は、マーシャリング実装でリスナーを別々に追加または削除できるようにするために、SystemManager によって送出されます。SystemManager によって送出されると、listener プロパティは null 以外になります。

EventListenerRequest

()コンストラクター
public function EventListenerRequest(type:String, bubbles:Boolean = false, cancelable:Boolean = true, eventType:String = null, listener:Function = null, useCapture:Boolean = false, priority:int = 0, useWeakReference:Boolean = false)

言語バージョン: ActionScript 3.0
製品バージョン: Flex 3
ランタイムバージョン: Flash Player 9, AIR 1.1

イベントリスナーを追加または削除する新しい要求を作成します。

パラメーター
type:String — イベントタイプ。イベントの基となるアクションを示します。EventListenerRequest.ADD または EventListenerRequest.REMOVE。
 
bubbles:Boolean (default = false) — イベントが表示リスト階層を上方にバブルできるかどうかを指定します。
 
cancelable:Boolean (default = true) — イベントに関連付けられた動作をキャンセルできるかどうかを指定します。
 
eventType:String (default = null) — 一般に addEventListener() メソッドに渡すメッセージのタイプです。
 
listener:Function (default = null) — リスナーが、キャプチャ段階、またはターゲットおよびバブリング段階で動作するかどうかを判断します。
 
useCapture:Boolean (default = false) — イベントリスナーの優先度レベルです。優先度は、32 ビット整数で指定します。
 
priority:int (default = 0) — リスナーへの参照が強参照と弱参照のいずれであるかを判断します。
 
useWeakReference:Boolean (default = false)

関連する API エレメント
