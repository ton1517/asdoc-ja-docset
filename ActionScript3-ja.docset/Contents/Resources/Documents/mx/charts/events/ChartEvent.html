クラスpublic class ChartEvent
継承ChartEvent MouseEvent Event Object

言語バージョン: ActionScript 3.0
製品バージョン: Flex 3
ランタイムバージョン: Flash Player 9, AIR 1.1

ChartEvent クラスは、グラフがクリックされた場合などの、グラフコントロールに特有のイベントを表します。 このイベントは、ChartItem オブジェクトがマウスでポイントされていない場合にのみ発生します。
