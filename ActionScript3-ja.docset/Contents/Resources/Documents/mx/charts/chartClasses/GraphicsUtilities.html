colorFromFill

()メソッド
public static function colorFromFill(f:Object):uint

言語バージョン: ActionScript 3.0
製品バージョン: Flex 3
ランタイムバージョン: Flash Player 9, AIR 1.1

塗りの値を単色のカラーに変換します。この便利なメソッドでは、画面上の Fill に最も近似した Fill からカラー値が取り出されます。

パラメーター

f:Object — カラーの抽出元の Fill オブジェクトです。

戻り値
uint — 塗りを表すカラー値です。

drawArc

()メソッド 
public static function drawArc(g:Graphics, x:Number, y:Number, startAngle:Number, arc:Number, radius:Number, yRadius:Number = NaN, continueFlag:Boolean = false):void

言語バージョン: ActionScript 3.0
製品バージョン: Flex 3
ランタイムバージョン: Flash Player 9, AIR 1.1

ターゲット Graphics オブジェクトに円弧を描画します。

パラメーター

g:Graphics — 描画先の Graphics オブジェクトです。
 
x:Number — 円弧の水平方向の原点です。
 
y:Number — 円弧の垂直方向の原点です。
 
startAngle:Number — 円弧の開始角度（ラジアン単位）です。
 
arc:Number — 円弧の長さ（ラジアン単位）です。
 
radius:Number — 円弧の水平方向の半径（ピクセル単位）です。
 
yRadius:Number (default = NaN) — 円弧の垂直方向の半径（ピクセル単位）です。指定しない場合は、半径と同じであると見なされます。
 
continueFlag:Boolean (default = false) — このルーチンで、描画前にグラフィックペンツールを円弧の始点に移動する必要があるかどうかを示します。処理中の線の描画を続行する場合は、true に設定します。

drawPolyLine

()メソッド 
public static function drawPolyLine(g:Graphics, pts:Array, start:int, end:int, hProp:String, vProp:String, stroke:IStroke, form:Object, moveToStart:Boolean = true):void

言語バージョン: ActionScript 3.0
製品バージョン: Flex 3
ランタイムバージョン: Flash Player 9, AIR 1.1

様々な形式の 1 つを使用してデータポイントを接続する線を描画します。

パラメーター

g:Graphics — 描画先の Graphics オブジェクトです。
 
pts:Array — 線を定義するデータポイントの配列です。
 
start:int — 線の描画時に使用する pts 配列の最初のポイントのインデックスです。
 
end:int — 線の描画時に使用する最後のポイントのインデックスです（このポイントは含まれません）。
 
hProp:String — 線のポイントの水平位置を定義する pts 配列に含まれているオブジェクトのプロパティの名前です。
 
vProp:String — 線のポイントの垂直位置を定義する pts 配列に含まれているオブジェクトのプロパティの名前です。
 
stroke:IStroke — 線のレンダリングに使用する Stroke です。
 
form:Object — 線をレンダリングする形式です。有効な値は、"segment"、"horizontal"、"vertical"、"step"、"reverseStep" または "curve" です。これらの形式は、LineSeries クラスの form プロパティに似ています。
 
moveToStart:Boolean (default = true) — 描画前にペンツールを線の始点に移動するかどうかを示します。この関数を使用して線の処理を続行する場合は、false を渡します。

fillFromStyle

()メソッド 
public static function fillFromStyle(v:Object):IFill

言語バージョン: ActionScript 3.0
製品バージョン: Flex 3
ランタイムバージョン: Flash Player 9, AIR 1.1

スタイル値を Fill オブジェクトに変換します。この便利なメソッドでは、スタイルによって割り当てられた値が Fill オブジェクトに変換されます。このオブジェクトは、画面の領域を塗りつぶすために使用できます。値が数値の場合は、対応する SolidColor に変換されます。

パラメーター

v:Object — Fill に変換する値です。

戻り値
IFill — 対応する IFill オブジェクトです。

fillRect

()メソッド 
public static function fillRect(g:Graphics, left:Number, top:Number, right:Number, bottom:Number, fill:Object = null, ls:Object = null):void

言語バージョン: ActionScript 3.0
製品バージョン: Flex 3
ランタイムバージョン: Flash Player 9, AIR 1.1

指定された IFill および IStroke オブジェクトを使用して矩形を塗りつぶします。このユーティリティ関数は、指定した Fill および Stroke を使用して、Graphics オブジェクト内の指定した矩形を塗りつぶします。Stroke を指定しない場合、矩形は境界線なしで塗りつぶされます。Fill を指定しない場合、矩形は塗りつぶしなしで描画されます。

パラメーター

g:Graphics — 描画先の Graphics オブジェクトです。
 
left:Number — 塗りつぶす矩形の左端です。
 
top:Number — 塗りつぶす矩形の上端です。
 
right:Number — 塗りつぶす矩形の右端です。
 
bottom:Number — 塗りつぶす矩形の下端です。
 
fill:Object (default = null) — 矩形の塗りつぶしに使用する IFill オブジェクトです。Fill を使用せずに矩形を描画する場合は、このパラメーターを null に設定します。
 
ls:Object (default = null) — 矩形の描画に使用する Stroke オブジェクトです。Stroke を使用せずに矩形を描画する場合は、このパラメーターを null に設定します。

setLineStyle

()メソッド 
public static function setLineStyle(g:Graphics, ls:IStroke):void

言語バージョン: ActionScript 3.0
製品バージョン: Flex 3
ランタイムバージョン: Flash Player 9, AIR 1.1

指定した Graphics オブジェクトに Stroke の値を適用します。このユーティリティ関数は、指定した Graphics オブジェクトに、渡された Stroke を適用するか、ls パラメーターが null に設定されている場合は、Graphics オブジェクトの線のスタイルをクリアします。

パラメーター

g:Graphics — 変更する Graphics オブジェクトです。
 
ls:IStroke — 適用する IStroke インスタンスです。Stroke なしでレンダリングする場合は null に設定します。
