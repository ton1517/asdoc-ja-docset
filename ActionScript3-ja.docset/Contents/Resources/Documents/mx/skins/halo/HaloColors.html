addHaloColors

()メソッド
public static function addHaloColors(colors:Object, themeColor:uint, fillColor0:uint, fillColor1:uint):void

言語バージョン: ActionScript 3.0
製品バージョン: Flex 3
ランタイムバージョン: Flash Player 9, AIR 1.1

斜めのエッジのカラーなど、Halo テーマをサポートするコンポーネントで使用されるカラーを計算します。 このメソッドは themeColor プロパティおよび fillColors プロパティを使用してそのカラーを計算します。

パラメーター

colors:Object — 計算されたカラー値が保存される対象のオブジェクトです。
 
themeColor:uint — themeColor スタイルプロパティの値です。
 
fillColor0:uint — 塗りの開始カラーです。
 
fillColor1:uint — 塗りの終了カラーです。

getCacheKey

()メソッド 
public static function getCacheKey(... colors):String

言語バージョン: ActionScript 3.0
製品バージョン: Flex 3
ランタイムバージョン: Flash Player 9, AIR 1.1

渡されたカラーに基づいて、固有のハッシュキーを返します。このキーを使用して、算出されたカラーが保存されるため、計算が 1 回で済みます。

パラメーター

... colors — uint 値で表記される RGB カラーの任意の数値です (0xFF0000 など)。

戻り値
String
