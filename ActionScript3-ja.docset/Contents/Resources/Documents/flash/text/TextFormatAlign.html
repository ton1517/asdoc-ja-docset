END

定数 
public static const END:String = "end"

言語バージョン: ActionScript 3.0

定数です。テキストを行の終端に行揃えします。左から右読みの言語での RIGHT、および右から左読みの言語での LEFT と同様です。

END 定数は、StageText クラスでのみ使用できます。

関連する API エレメント
