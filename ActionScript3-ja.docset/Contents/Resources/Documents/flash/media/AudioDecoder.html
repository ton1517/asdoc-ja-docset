パッケージflash.media
クラスpublic final class AudioDecoder
継承AudioDecoder Object

言語バージョン: ActionScript 3.0
ランタイムバージョン: Flash Player 11, AIR 3

AudioDecoder クラスは、システムでサポートできるマルチチャンネルオーディオのタイプを列挙します。

このクラスに定義されている定数の 1 つを、Capabilities クラスの hasMultiChannelAudio() メソッドのパラメーターとして使用します。

AIR プロファイルのサポート：マルチチャンネルオーディオはテレビ用 AIR デバイスでのみサポートされます。他のすべてのデバイスでは、hasMultiChannelAudio() は常に false を返します。複数のプロファイル間での API サポートについて詳しくは、AIR プロファイルのサポートを参照してください。
