クラスpublic final class RectangleTexture
継承RectangleTexture TextureBase EventDispatcher Object

言語バージョン: ActionScript 3.0
ランタイムバージョン: Flash Player 11.6, AIR 3.6

RectangleTexture クラスは、レンダリングコンテキストにアップロードされる 2D テクスチャを表します。

レンダリング時に使用するための 2D テクスチャを定義しします。

Texture を直接インスタンス化することはできません。Context3D createRectangleTexture() メソッドを使用して、インスタンスを作成します。

関連する API エレメント

flash.display3D.Context3D.createRectangleTexture()
flash.display3D.Context3D.setTextureAt()

uploadFromBitmapData

()メソッド
public function uploadFromBitmapData(source:BitmapData):void

言語バージョン: ActionScript 3.0
ランタイムバージョン: Flash Player 11.6, AIR 3.6

BitmapData オブジェクトからテクスチャをアップロードします。

パラメーター

source:BitmapData — ビットマップ。

例外
TypeError — Null ポインターエラーです：source が null の場合。
 
ArgumentError — ミップレベルが大きすぎます：指定したミップレベルが、Texture のサイズのいずれか大きいほうによって示されるレベルを超える場合。
 
ArgumentError — BitmapData が無効です：source の BitmapData オブジェクトに有効な立方体テクスチャ面が含まれていない場合。イメージには、2 の累乗に等しい数の側面と、指定されたミップレベルにおける正しいサイズが設定されている必要があります。

uploadFromByteArray

()メソッド 
public function uploadFromByteArray(data:ByteArray, byteArrayOffset:uint):void

言語バージョン: ActionScript 3.0
ランタイムバージョン: Flash Player 11.6, AIR 3.6

ByteArray からテクスチャをアップロードします。

パラメーター

data:ByteArray — テクスチャの塗りに必要なバイト数をテクスチャの内部形式で含むバイト配列。RGBA テクスチャは、テクセルコンポーネントごとのバイト数（1 または 4）として読み取られます。浮動小数点テクスチャは、テクセルコンポーネントごとの浮動小数点数（1 または 4）として読み取られます。ByteArray オブジェクトではリトルエンディアン形式を使用する必要があります。
 
byteArrayOffset:uint — バイト配列オブジェクト内でテクスチャデータの読み取りを開始する位置。

例外
TypeError — Null ポインターエラーです：data が null の場合。
 
ArgumentError — ミップレベルが大きすぎます：指定したミップレベルが、Texture のサイズのいずれか大きいほうによって示されるレベルを超える場合。
 
RangeError — 入力サイズが正しくありません：byteArrayOffset から data バイト配列の末尾までのバイト数が、そのミップレベルのテクスチャに必要とされるデータ量よりも少ない場合。
