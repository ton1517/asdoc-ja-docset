flash.system パッケージには、セキュリティやガベージコレクションなどのシステムレベルの機能にアクセスするためのクラスが含まれます。


関数
 関数説明
 fscommand SWF ファイルが、Flash Player または Flash Player のホストプログラム（Web ブラウザーなど）と通信できるようになります。
クラス
 クラス説明
 ApplicationDomain ApplicationDomain クラスは、クラス定義の個別のグループのコンテナです。
 Capabilities Capabilities クラスは、アプリケーションをホスティングするシステムとランタイムを記述するプロパティを提供します。
 ImageDecodingPolicy     ImageDecodingPolicy クラスは、LoaderContext クラス内の imageDecodingPolicy の値を提供します。
 IME IME クラスを使用すると、クライアントコンピューターで実行されている Flash ランタイムアプリケーションで、オペレーティングシステムの IME を直接操作できます。
 IMEConversionMode このクラスには、IME.conversionMode プロパティで使用できる定数があります。
 JPEGLoaderContext JPEGLoaderContext クラスには、JPEG 画像のロード時に非ブロックフィルターを有効にするプロパティが含まれています。
 LoaderContext LoaderContext クラスは、Loader クラスを使用して、SWF ファイルなどのメディアをロードするオプションを指定します。
 MessageChannel MessageChannel クラスは、ワーカーが別のワーカーと通信するためのメカニズムを提供します。
 MessageChannelState このクラスによって、MessageChannel クラスの state プロパティの有効な値を表す定数が定義されます。
 Security Security クラスによって、異なるドメインに属するコンテンツが互いにどのようにやり取りするかを指定できます。
 SecurityDomain SecurityDomain クラスは、現在のセキュリティ "サンドボックス" を表し、セキュリティドメインとも呼ばれます。
 SecurityPanel SecurityPanel クラスは、どのセキュリティ設定パネルを表示するかを指定する値を提供します。
 System System クラスには、ローカル設定と操作に関連するプロパティが含まれています。
 SystemUpdater SystemUpdater クラスを使用すると、Adobe Access 用の DRM モジュールなどの Flash Player のモジュールや、Flash Player 自体を更新できます。
 SystemUpdaterType SystemUpdaterType クラスは、システムの更新に使用される定数を提供します。
 TouchscreenType TouchscreenType クラスは、様々な種類のタッチスクリーンを表す値を提供する列挙クラスです。
 Worker Worker オブジェクトは、Flash ランタイムの仮想インスタンスである、ワーカーを表します。
 WorkerDomain WorkerDomain クラスは、Worker オブジェクトを作成し、そのオブジェクトにアクセスするための方法を提供します。
 WorkerState このクラスによって、Worker クラスの state プロパティの有効な値を表す定数が定義されます。
