Accelerometer クラスは、デバイスにインストールされている加速度センサーから加速度の更新情報を取得したときに、AccelerometerEvent オブジェクトを送出します。

プロパティの詳細

accelerationX

プロパティ
accelerationX:Number

言語バージョン: ActionScript 3.0
ランタイムバージョン: AIR 2, Flash Player 10.1, Flash Lite 4

G（重力加速度）で測定した X 軸加速度です。（1 G は約 9.8 m/秒/秒です）。デバイスが直立状態にある場合、X 軸の方向はデバイスの左から右です。デバイスが右方向に移動する場合、加速度は正の値になります。

accelerationY

プロパティ 
accelerationY:Number

言語バージョン: ActionScript 3.0
ランタイムバージョン: AIR 2, Flash Player 10.1, Flash Lite 4

G（重力加速度）で測定した Y 軸加速度です。（1 G は約 9.8 m/秒/秒です）。デバイスが直立状態にある場合、Y 軸の方向は下から上です。デバイスがこの軸を基準にして上方向に移動する場合、加速度は正の値になります。



実装
    public function get accelerationY():Number
    public function set accelerationY(value:Number):void

accelerationZ

プロパティ 
accelerationZ:Number

言語バージョン: ActionScript 3.0
ランタイムバージョン: AIR 2, Flash Player 10.1, Flash Lite 4

G（重力加速度）で測定した Z 軸加速度です。（1 G は約 9.8 m/秒/秒です）。Z 軸は、デバイスの表面に対して垂直です。表面がより高い位置になるようにデバイスを移動する場合、加速度は正の値になります。



実装
    public function get accelerationZ():Number
    public function set accelerationZ(value:Number):void

timestamp

プロパティ 
timestamp:Number

言語バージョン: ActionScript 3.0
ランタイムバージョン: AIR 2, Flash Player 10.1, Flash Lite 4

ランタイムが初期化されてからイベント発生時までの時間（秒）です。例えば、アプリケーションの初期化から 4 秒後に、デバイスが加速度センサーのデータをキャプチャした場合、このイベントの timestamp プロパティは 4000 に設定されます。

コンストラクターの詳細

AccelerometerEvent

()コンストラクター
public function AccelerometerEvent(type:String, bubbles:Boolean = false, cancelable:Boolean = false, timestamp:Number = 0, accelerationX:Number = 0, accelerationY:Number = 0, accelerationZ:Number = 0)

言語バージョン: ActionScript 3.0
ランタイムバージョン: AIR 2, Flash Player 10.1, Flash Lite 4

3 軸加速度に関する情報を含む AccelerometerEvent オブジェクトを作成します。イベントリスナーには Event オブジェクトがパラメーターとして渡されます。

パラメーター
type:String — イベントのタイプです。イベントリスナーは、継承された type プロパティを通じてこの情報にアクセスできます。update イベントのタイプは AccelerometerEvent.UPDATE のみです。
 
bubbles:Boolean (default = false) — Event オブジェクトでバブリングを実行できるかどうかを判断します。イベントリスナーは、継承された bubbles プロパティを通じてこの情報にアクセスできます。
 
cancelable:Boolean (default = false) — Event オブジェクトがキャンセル可能かどうかを判断します。イベントリスナーは、継承された cancelable プロパティを通じてこの情報にアクセスできます。
 
timestamp:Number (default = 0) — 加速度センサーの更新のタイムスタンプです。
 
accelerationX:Number (default = 0) — G（9.8m/秒/秒）の X 軸の加速度値。
 
accelerationY:Number (default = 0) — G（9.8m/秒/秒）の Y 軸の加速度値。
 
accelerationZ:Number (default = 0) — G（9.8m/秒/秒）の Z 軸の加速度値。
