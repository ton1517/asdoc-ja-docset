言語バージョン: ActionScript 3.0
ランタイムバージョン: AIR 2

DNSResolverEvent クラスは、DNS（Domain Name System）ルックアップ操作の結果を表します。

DNS クエリを開始するには、DNSResolver lookup() メソッドを使用します。クエリによって返されるリソースレコードは、この DNSResolverEvent オブジェクトの resourceRecords 配列に置かれます。

public function DNSResolverEvent(type:String, bubbles:Boolean = false, cancelable:Boolean = false, host:String = "", resourceRecords:Array = null)

言語バージョン: ActionScript 3.0
ランタイムバージョン: AIR 2

DNS ルックアップ操作の結果を含む DNSResolverEvent オブジェクトを作成します。イベントリスナーには Event オブジェクトがパラメーターとして渡されます。

パラメーター
type:String — イベントのタイプです。有効な値は次のとおりです。DNSResolverEvent.LOOKUP
 
bubbles:Boolean (default = false) — Event オブジェクトがイベントフローのバブリング段階で処理されるかどうかを判断します。
 
cancelable:Boolean (default = false) — Event オブジェクトがキャンセル可能かどうかを判断します。
 
host:String (default = "") — 新規イベントが応答する DNSResolver クラスの lookup() メソッドの呼び出しで使用されるホスト名、IP アドレスまたはサービスロケーターなどのクエリ文字列です。
 
resourceRecords:Array (default = null) — 返された DNS リソースレコードのリストです。
