パッケージflash.events
クラスpublic class VideoEvent
継承VideoEvent Event Object

言語バージョン: ActionScript 3.0
ランタイムバージョン: Flash Player 10.2, AIR 3

このイベントクラスは、現在のビデオレンダリングステータスをレポートします。このイベントは、次の目的で使用します。

  • ビデオの表示領域がいつ変更または初期化されるのかを検知するため。サイズ変更をポーリングする代わりにこのイベントを使用します。このイベントを受け取ったとき、Video.videoSize および Video.videoHeight にアクセスして現在再生されているビデオのピクセル範囲を取得することができます。
  • ビデオがソフトウェアまたは GPU でデコードされているかどうかを確認するため。status プロパティが「accelerated」を返した場合は、可能であれば StageVideo クラスを使用してください。

パラメーター
type:String — イベントのタイプです。有効な値は、VideoEvent.RENDER_STATE です。
 
bubbles:Boolean (default = false) — この Event オブジェクトがイベントフローのバブリング段階で処理されるかどうかを示します。
 
cancelable:Boolean (default = false) — このイベントをトリガーしたアクションをキャンセルできるかどうかを示します。
 
status:String (default = null) — ビデオのレンダリングの状態です。
