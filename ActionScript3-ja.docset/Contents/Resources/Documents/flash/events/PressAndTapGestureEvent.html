クラスpublic class PressAndTapGestureEvent
継承PressAndTapGestureEvent GestureEvent Event Object

言語バージョン: ActionScript 3.0
ランタイムバージョン: Flash Player 10.1, AIR 2, Flash Lite 4

PressAndTapGestureEvent クラスにより、タッチ対応デバイスのプレスアンドタップジェスチャを処理することができます。InteractiveObject クラスからプロパティを継承するオブジェクトは、送出されたイベントオブジェクトの最初のタッチポイント（プレス）および 2 つ目のタッチポイント（タップ）をキャプチャします。 プレスアンドタップジェスチャは、通常、状況依存ポップアップメニューを開くときに使用します。

public function PressAndTapGestureEvent(type:String, bubbles:Boolean = true, cancelable:Boolean = false, phase:String = null, localX:Number = 0, localY:Number = 0, tapLocalX:Number = 0, tapLocalY:Number = 0, ctrlKey:Boolean = false, altKey:Boolean = false, shiftKey:Boolean = false, commandKey:Boolean = false, controlKey:Boolean = false)

言語バージョン: ActionScript 3.0
ランタイムバージョン: Flash Player 10.1, AIR 2, Flash Lite 4

複雑なマルチタッチイベント（ユーザーが状況依存ポップアップメニューを開くなど）に関する情報を含む Event オブジェクトを作成します。イベントリスナーには Event オブジェクトがパラメーターとして渡されます。

パラメーター
type:String — イベントのタイプ：PressAndTapGestureEvent.GESTURE_PRESS_AND_TAP。
 
bubbles:Boolean (default = true) — Event オブジェクトがイベントフローのバブリング段階で処理されるかどうかを判断します。
 
cancelable:Boolean (default = false) — Event オブジェクトがキャンセル可能かどうかを判断します。
 
phase:String (default = null) — この値は、タッチジェスチャの開始、進行、終了を追跡します。有効な値は次のとおりです。GesturePhase.BEGIN、GesturePhase.END、GesturePhase.UPDATE、または GesturePhase.ALL。
 
localX:Number (default = 0) — 表示オブジェクトを基準とするイベント発生位置の水平座標です。
 
localY:Number (default = 0) — 表示オブジェクトを基準とするイベント発生位置の垂直座標です。
 
tapLocalX:Number (default = 0) — インタラクティブオブジェクトを基準とするイベント発生位置の水平座標です。
 
tapLocalY:Number (default = 0) — インタラクティブオブジェクトを基準とするイベント発生位置の垂直座標です。
 
ctrlKey:Boolean (default = false) — Windows または Linux では、Ctrl キーがアクティブになっているかどうかを示します。Macintosh では、Ctrl キーと Command キーのいずれがアクティブになっているかどうかを示します。
 
altKey:Boolean (default = false) — Alt キーがアクティブになっているかどうかを示します（Windows または Linux のみ）。
 
shiftKey:Boolean (default = false) — Shift キーがアクティブになっているかどうかを示します。
 
commandKey:Boolean (default = false) — （AIR のみ）Command キーがアクティブになっているかどうかを示します（Macintosh のみ）。このパラメーターは Adobe AIR 専用です。Flash Player コンテンツに対しては設定しないでください。
 
controlKey:Boolean (default = false) — （AIR のみ）Control キーまたは Ctrl キーがアクティブになっているかどうかを示します。このパラメーターは Adobe AIR 専用です。Flash Player コンテンツに対しては設定しないでください。

type プロパティ（GESTURE_PRESS_AND_TAP タッチイベントオブジェクト）の値を定義します。

PressAndTapGestureEvent オブジェクトには次のプロパティがあります。

プロパティ値
altKeyAlt キーがアクティブになっている場合は true（Windows または Linux）。
bubblestrue
cancelablefalse は、キャンセルするデフォルトの動作がないことを示します。
commandKeyMacintosh で Command キーがアクティブになっている場合は true、非アクティブになっている場合は false です。 Windows では常に false になります。
controlKeyCtrl キーまたは Control キーがアクティブになっている場合は true、非アクティブになっている場合は false です。
ctrlKeyWindows または Linux では Ctrl キーがアクティブになっている場合は、true です。Macintosh では Ctrl キーまたは Command キーがアクティブになっている場合は、true です。それ以外の場合は false です。
currentTargetイベントリスナーで Event オブジェクトをアクティブに処理しているオブジェクトです。
eventPhaseイベントがオブジェクト階層を通過するときの最新の段階（イベントがキャプチャ（1）、ターゲット（2）、またはバブリング（3）の段階にあることを示す数値）。
localX表示オブジェクトを基準とするイベント発生位置の水平座標です。
localY表示オブジェクトを基準とするイベント発生位置の垂直座標です。
phaseイベントフローの現在の段階であり、GesturePhase クラスから取得される値です。有効な値は次のとおりです。GesturePhase.BEGIN、GesturePhase.UPDATE、GesturePhase.END、または GesturePhase.ALL。プレスアンドタップジェスチャによって、GesturePhase.BEGIN、GesturePhase.UPDATE、GesturePhase.END シーケンスが生成されるか、または単一の GesturePhase.ALL 段階が生成されます。
shiftKeyShift キーがアクティブになっている場合は true、非アクティブになっている場合は false です。
stageXグローバルステージ座標を基準とするイベント発生位置の水平座標です。
stageYグローバルステージ座標を基準とするイベント発生位置の垂直座標です。
tapLocalXインタラクティブオブジェクトを基準とするイベント発生位置の水平座標です。
tapLocalYインタラクティブオブジェクトを基準とするイベント発生位置の垂直座標です。
tapStageXグローバルステージ座標を基準とするタップタッチ発生位置の水平座標です。
tapStageYグローバルステージ座標を基準とするタップタッチ発生位置の垂直座標です。
targetタッチデバイスの下にある InteractiveObject インスタンスです。target は、必ずしもイベントリスナーを登録した表示リスト内のオブジェクトとは限りません。現在イベントを処理している表示リスト内のオブジェクトにアクセスするには、currentTarget プロパティを使用します。

例 この例の使用方法
GesturePressAndTap.as

次の例は、GESTURE_PRESS_AND_TAP イベントのイベント処理を示しています。 ユーザーがプレスアンドタップジェスチャを実行しているときに、mySprite が回転し、myTextField に現在の段階が設定されます。
コピー

Multitouch.inputMode = MultitouchInputMode.GESTURE;

var mySprite = new Sprite();
mySprite.addEventListener(PressAndTapGestureEvent.GESTURE_PRESS_AND_TAP , onPressAndTap );
mySprite.graphics.beginFill(0x336699);
mySprite.graphics.drawRect(0, 0, 100, 80);
var myTextField = new TextField();
myTextField.y = 200;
addChild(mySprite);
addChild(myTextField);

function onPressAndTap(evt:PressAndTapGestureEvent):void {

    evt.target.rotation -= 45;

    if (evt.phase==GesturePhase.BEGIN) {
        myTextField.text = "Begin";
    }
    if (evt.phase==GesturePhase.UPDATE) {
        myTextField.text = "Update";
    }
    if (evt.phase==GesturePhase.END) {
        myTextField.text = "End";
    }
}
