クラスpublic final class GesturePhase
継承GesturePhase Object

言語バージョン: ActionScript 3.0
ランタイムバージョン: Flash Player 10.1, AIR 2, Flash Lite 4

GesturePhase クラスは、定数値の列挙クラスで、GestureEvent、PressAndTapGestureEvent および TransformGestureEvent クラスと共に使用します。これらの値を使用して、タッチジェスチャ（タッチ対応スクリーン上で複数の指を動かすなど）の開始、進行、終了を追跡し、アプリケーションがユーザーの接触動作の各段階に応答できるようにします。一部のジェスチャ（スワイプおよび 2 本の指によるタップジェスチャ）には複数の段階がないので、イベントオブジェクトの phase プロパティを all に設定します。

例 この例の使用方法
TransformGestureEventExample.as

次の例は、GESTURE_ROTATE イベントのイベント処理を示しています。ユーザーがタッチ対応デバイスで回転ジェスチャを実行しているときに、mySprite が回転し、myTextField に現在の段階が設定されます。
コピー

Multitouch.inputMode = MultitouchInputMode.GESTURE;

var mySprite = new Sprite();
mySprite.addEventListener(TransformGestureEvent.GESTURE_ROTATE , onRotate );
mySprite.graphics.beginFill(0x336699);
mySprite.graphics.drawRect(0, 0, 100, 80);
var myTextField = new TextField();
myTextField.y = 200;
addChild(mySprite);
addChild(myTextField);

function onRotate(evt:TransformGestureEvent):void {

    evt.target.rotation -= 45;

    if (evt.phase==GesturePhase.BEGIN) {
        myTextField.text = "Begin";
    }
    if (evt.phase==GesturePhase.UPDATE) {
        myTextField.text = "Update";
    }
    if (evt.phase==GesturePhase.END) {
        myTextField.text = "End";
    }
}
