クラスpublic class DRMDeviceGroupEvent
継承DRMDeviceGroupEvent Event Object

言語バージョン: ActionScript 3.0
ランタイムバージョン: AIR 3.0

デバイスグループ関連の呼び出しが正常に完了した場合に DRMManager によって発行されます。DRMManager.addToDeviceGroup() メソッドを使用して、1 つのデバイスグループに複数のデバイスを登録できます。指定されたコンテンツについて、ドメインにバインドされた有効なバウチャーを持つデバイスがある場合、アプリケーションは、DRMVoucher.toByteArray() メソッドを使用して、直列化された DRM バウチャーを抽出できます。

コンテンツメタデータでドメイン登録が必要であると指定されている場合は、アプリケーションでデバイスグループに参加するための API を呼び出すことができます。この操作は、ドメインサーバーに送信されるドメイン登録リクエストをトリガーします。ライセンスがデバイスグループに発行されると、デバイスグループに参加している他のデバイスにライセンスを書き出したり共有したりできます。
