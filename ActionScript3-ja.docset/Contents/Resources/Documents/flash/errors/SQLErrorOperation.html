パッケージflash.errors
クラスpublic class SQLErrorOperation
継承SQLErrorOperation Object

言語バージョン: ActionScript 3.0
ランタイムバージョン: AIR 1.0

このクラスには、SQLError.operation プロパティの有効な値を表す定数が含まれます。これらの値は、エラー発生の原因となった試行された操作を示します。

それぞれの値は、SQLConnection クラスまたは SQLStatement クラスの操作の 1 つを表します。

関連する API エレメント

flash.errors.SQLError.operation
flash.data.SQLConnection
flash.data.SQLStatement
