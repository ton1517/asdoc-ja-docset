flash.geom パッケージには、BitmapData クラスとビットマップキャッシュ機能をサポートするポイント、矩形、変換マトリックスなどのジオメトリクラスが含まれます。


クラス
 クラス説明
 ColorTransform ColorTransform クラスを使用すると、表示オブジェクトのカラー値を調整することができます。
 Matrix Matrix クラスは、2 つの座標空間の間におけるポイントのマッピング方法を決定する変換マトリックスを表します。
 Matrix3D Matrix3D クラスは、3 次元（3D）表示オブジェクトの位置と方向を決定する変換マトリックスを表します。
 Orientation3D Orientation3D クラスは、Matrix3D オブジェクトの方向スタイルを表すための定数値の列挙です。
 PerspectiveProjection PerspectiveProjection クラスを使用すると、遠近法に基づく変形を表示オブジェクトおよびそのオブジェクトのすべての子に簡単に割り当てることができます。
 Point Point オブジェクトは 2 次元の座標系の位置を表します。x は水平方向の軸を表し、y は垂直方向の軸を表します。
 Rectangle Rectangle オブジェクトは、その位置（左上隅のポイント (x, y) で示される）、および幅と高さで定義される領域です。
 Transform Transform クラスは、表示オブジェクトに適用されるカラー調整プロパティと 2 次元または 3 次元の変換オブジェクトへのアクセスを提供します。
 Utils3D Utils3D クラスには、特定の 3 次元マトリックス操作の実装を単純化する静的メソッドが含まれます。
 Vector3D Vector3D クラスは、極座標 x、y、および z を使用して 3 次元空間でのポイントまたは位置を表します。
