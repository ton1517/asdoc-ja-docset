createStageTemplate

()メソッド
public function createStageTemplate(stageTemplate:com.adobe.solutions.rca.vo:StageTemplateVO, overwrite:Boolean):mx.rpc:AsyncToken

言語バージョン: ActionScript 3.0
製品バージョン: Review, Commenting, and Approval Building Block 10
ランタイムバージョン: Flash Player 10.2, AIR (unsupported)

This operation creates a new stage template in the system.

パラメーター

stageTemplate:com.adobe.solutions.rca.vo:StageTemplateVO — StageTemplateVO to be created.
 
overwrite:Boolean — A Boolean value for error-proof overwriting of stage templates. If the value is True and a template exists with same name, then the existing template is replaced with the provided stageTemplate.

戻り値
mx.rpc:AsyncToken — It returns a token on which success or failure handlers can be attached. The resulting event contains the created stageTemplateVO.

This operation deletes a stage template from the system. If this stage is referred by one (or more) ReviewTemplateVO, delete operation fails with an appropriate exception.

パラメーター

templateName:String — The unique stage template name.

戻り値
mx.rpc:AsyncToken — It returns a token on which success or failure handlers can be attached. The resulting event contains the deleted StageTemplateVO.
