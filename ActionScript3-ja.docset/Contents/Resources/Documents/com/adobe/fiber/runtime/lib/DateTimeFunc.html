dateAdd

()メソッド 
public static function dateAdd(part:String, units:int, date:Date):Date

言語バージョン: ActionScript 3.0
製品バージョン: Adobe Digital Enterprise Platform Data Services for Java EE 3
ランタイムバージョン: Flash Player 9, AIR 1.1

日付に時間の単位を追加します。次の単位があります。

  • yyyy：年
  • q：四半期
  • m：月
  • y：年間通算日
  • d：日
  • w：曜日
  • ww：週
  • h：時
  • n：分
  • s：秒
  • l：ミリ秒

パラメーター

part:String — 追加する単位の種類
 
units:int — 単位の数値
 
date:Date — 日付

戻り値
Date — 新しい日付。日付が null の場合は null

dateCompare

()メソッド 
public static function dateCompare(date1:Date, date2:Date):int

言語バージョン: ActionScript 3.0
製品バージョン: Adobe Digital Enterprise Platform Data Services for Java EE 3
ランタイムバージョン: Flash Player 9, AIR 1.1

2 つの日付を使用し、最初の日付が後の日付より小さい場合は -1、等しい場合は 0、大きい場合は 1 を返します。

パラメーター

date1:Date — 比較対象の 1 つ目の日付
 
date2:Date — 比較対象の 2 つ目の日付

戻り値
int — date-1 < date2 の場合 int -1、date1=date2 の場合 int 0、date 1 > date2 の場合 int 1。いずれかの日付が null の場合は、0 を返します。
