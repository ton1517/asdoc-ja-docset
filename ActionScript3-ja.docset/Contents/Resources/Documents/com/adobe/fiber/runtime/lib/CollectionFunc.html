part

()メソッド 
public static function part(value:Object, list:ArrayCollection):int

言語バージョン: ActionScript 3.0
製品バージョン: Adobe Digital Enterprise Platform Data Services for Java EE 3
ランタイムバージョン: Flash Player 9, AIR 1.1

値とソートする値リストの指定により、その値より小さいリストアイテムの数を返します。（ソートしない場合は、値より小さいアイテムから成るリストのプレフィックスの長さを返します。）

例えば、[10, 20, 30] を含むコレクション C を設定して part(20, C) を呼び出した場合、コレクションには 20 より小さい値が 1 つ（10）のみあるため、1 が返されます。

パラメーター

value:Object — 検索する値
 
list:ArrayCollection — ソートするコレクション

戻り値
int — リスト内で、指定された値より小さい値のエントリの数
