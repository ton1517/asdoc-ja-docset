DataServiceWrapper クラスは、DataService インスタンス用に生成されるサービスラッパーのスーパークラスです。このようなサービスラッパーは、LiveCycle Data Services に裏付けされたものとしてマークされたモデル内のすべてのエンティティに対して生成されます。ラッパーは、基礎にある DataService クラスの使用に利便性をもたらします。生成されるラッパーには、背後の LiveCycle Data Services Data Management Service の宛先がサポートする各入力内容（モデル内のフィルター）に対応する、厳密に型指定された個別の関数が含まれます。

ラップされたデータサービスの fault イベントは、ラッパーにより伝播されます。詳細については、DataService.fault を参照してください。

FAULT 定数により、fault イベントのイベントオブジェクトの type プロパティ値が定義されます。

イベントオブジェクトの各プロパティには次の値が設定されています。

プロパティ値
bubblesfalse
cancelabletrue。関連付けられたトークンの responder.fault メソッドから preventDefault() を呼び出すと、サービスまたは処理がこのイベントを送出しなくなります。
currentTargetイベント処理を行うイベントリスナーの定義用オブジェクトを表します。 例えば、イベントリスナーの登録に myButton.addEventListener() を使用すると、myButton の値は currentTarget となります。
fault 発生した競合を保持する Fault オブジェクトです。
item フォルトが発生したアイテムです。
identity フォルトが発生したアイテムの ID です。
message このイベントに関連付けられたメッセージ。
token メソッドの呼び出しを表すトークン。Asynchronous Completion Token パターンで使用されます。
targetイベントを送出したオブジェクトを表します。このオブジェクトは必ずしもイベントを受け取るオブジェクトとは限りません。 イベントを受け取るオブジェクトに常にアクセスするには、currentTarget プロパティを使用してください。

関連する API エレメント

mx.data.DataService
